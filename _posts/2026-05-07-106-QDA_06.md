# 6. Sample-based Krylov Quantum Diagonalization (SKQD)

샘플 기반 크릴로프 양자 대각화(Sample-based Krylov quantum diagonalization, SKQD)에 대한 이번 레슨은 이전 레슨들에서 설명된 방법들을 결합합니다. 이는 Qiskit 패턴 프레임워크를 활용하는 단일 예제로 구성되어 있습니다:

* 1단계: 문제를 양자 회로 및 연산자로 매핑
* 2단계: 대상 하드웨어에 맞게 최적화
* 3단계: Qiskit Primitives를 사용하여 실행
* 4단계: 후처리

샘플 기반 양자 대각화 방법에서 중요한 단계는 서브스페이스(부분공간)를 위한 고품질 벡터를 생성하는 것입니다. 이전 레슨에서는 화학 해밀토니안을 위한 서브스페이스 벡터를 생성하기 위해 LUCJ 안사츠(ansatz)를 사용했습니다. 이번 레슨에서는 레슨 2에서 논의했던 양자 크릴로프 상태$
\[1\]$를 사용할 것입니다. 먼저, 시간 진화(time evolution) 연산을 사용하여 양자 컴퓨터에서 크릴로프 공간을 생성하는 방법을 검토할 것입니다. 그런 다음 이 공간에서 샘플링을 수행할 것입니다. 시스템 해밀토니안을 샘플링된 서브스페이스에 투영하고 이를 대각화하여 바닥 상태 에너지를 추정할 것입니다. 이 알고리즘은 레슨 2에서 설명된 가정 하에 바닥 상태로 증명 가능하고 효율적으로 수렴합니다.

## 0. 크릴로프 공간 (The Krylov space)

$r$차 크릴로프 공간 $\mathcal{K}^r$은 행렬 $A$의 거듭제곱(최대 $r-1$차까지)을 기준 벡터 $\vert v \rangle$에 곱하여 얻은 벡터들로 생성되는 공간임을 상기하십시오.
<p>
$$\mathcal{K}^r = \left\{ \vert v \rangle, A \vert v \rangle, A^2 \vert v \rangle, ..., A^{r-1} \vert v \rangle \right\}$$
</p>

행렬 $A$가 해밀토니안 $H$인 경우, 이에 대응하는 공간을 *거듭제곱 크릴로프 공간(power Krylov space)* $\mathcal{K}_P$라고 부릅니다. $A$가 해밀토니안에 의해 생성되는 시간 진화 연산자 $U=e^{-iH(dt)}$인 경우, 이 공간은 유니터리 크릴로프 공간(unitary Krylov space) $\mathcal{K}_U$라고 부릅니다. $H$는 유니터리 연산자가 아니기 때문에 양자 컴퓨터에서 거듭제곱 크릴로프 서브스페이스를 직접 생성할 수는 없습니다. 대신, 거듭제곱 크릴로프 공간과 유사한 [수렴 보장](https://arxiv.org/abs/2110.07492)을 제공하는 것으로 나타나는 시간 진화 연산자 $U = e^{-iH(dt)}$를 사용할 수 있습니다. 그러면 $U$의 거듭제곱은 서로 다른 시간 단계 $U^k = e^{-iH(k dt)}$가 되며, 여기서 $k = 0, 1, 2, ..., (r-1)$입니다.

<p>
$$\mathcal{K}_U^r = \left\{ \vert \psi \rangle, U \vert \psi \rangle, U^2 \vert \psi \rangle, ..., U^{r-1} \vert \psi \rangle \right\}$$
</p>

## 1. 문제를 양자 회로 및 연산자로 매핑

이번 수업에서는 주기 경계 조건을 갖는 $L = 22$개의 사이트를 가진 반강자성 XX-Z 스핀-1/2 사슬의 해밀토니안을 살펴봅니다.

<p>
$$
    H = \sum_{i, j}^{N} J_{xy} (X_{i} X_{j} + Y_{i} Y_{j}) + Z_{i} Z_{j}
$$
</p>

```python
from qiskit.transpiler import CouplingMap
from qiskit_addon_utils.problem_generators import generate_xyz_hamiltonian

num_spins = 22
coupling_map = CouplingMap.from_ring(num_spins)
H_op = generate_xyz_hamiltonian(coupling_map, coupling_constants=(0.3, 0.3, 1.0))
```

Krylov 공간을 구성하기 위해, 우리는 세 가지 주요 요소가 필요합니다:

1. Krylov 차원($r$) 및 시간 간격($dt$)의 선택.
2. 목표(바닥) 상태와 다항식 중첩을 가지는 초기(기준) 상태(위의 벡터 $\vert v \rangle$)로, 여기서 목표 상태는 희소합니다. 다항식 중첩에 대한 이 요구사항은 양자 위상 추정 알고리즘에서와 동일합니다.
3. 시간 전개 연산자 $U^{k}=e^{-iH(k * dt)}$ ($k = 0, 1, 2, ..., r-1$).

선택된 $r$ 값(및 $dt$)에 대해, 우리는 $r$개의 개별 양자 회로를 생성하고 이들로부터 샘플링할 것입니다. 각 양자 회로는 기준 상태의 양자 회로 표현과 $k$ 값에 대한 시간 전개 연산자를 결합하여 생성됩니다.

더 큰 Krylov 차원은 추정된 에너지의 수렴성을 향상시킵니다. 우리는 수렴 경향을 설명하기 위해 이 단원에서 차원을 $5$로 설정합니다.

참고문헌 $\[2\]$ (#references)는 KQD에 대해 충분히 작은 시간 간격이 $\pi / \vert \vert H \vert \vert$이며, 이 값을 과대평가하기보다는 과소평가하는 것이 바람직하다고 보여주었습니다. 반면에, $dt$를 너무 작게 선택하면 Krylov 기저 벡터들이 시간 간격마다 덜 달라지기 때문에 Krylov 부분공간의 조건이 나빠집니다. 또한, 이 $dt$ 선택이 SKQD의 수렴에 충분함이 증명적으로 입증되었지만, 이러한 샘플링 기반 맥락에서 실제 최적의 $dt$ 선택은 현재 진행 중인 연구 주제입니다. 이 단원에서는 $dt = 0.15$로 설정합니다.

Krylov 차원과 시간 간격 외에도, 시간 전개를 위한 트로터 단계 수를 설정해야 합니다. 단계 수가 너무 적으면 더 큰 트로터화 오류가 발생하고, 너무 많으면 더 깊은 회로로 이어집니다. 이 단원에서는 트로터 단계 수를 $6$으로 설정합니다.

```python
# Set parameters for quantum Krylov algorithm
krylov_dim = 5  # size of krylov subspace
dt = 0.15
num_trotter_steps = 6
```

다음으로, 기저 상태와 어느 정도 겹치는 기준 상태 $\vert \psi \rangle$를 선택해야 합니다. 이 해밀토니안의 경우, 1과 0이 교대로 나타나는 *Neel* 상태 $\vert ...101...010...101 \rangle$를 기준 상태로 사용합니다.

```python
# Prep `Neel` state as the reference state for evolution
from qiskit import QuantumCircuit

qc_state_prep = QuantumCircuit(num_spins)
for i in range(num_spins):
    if i % 2 == 0:
        qc_state_prep.x(i)
```

마지막으로 시간 진화 연산자를 양자 회로에 매핑해야 합니다. 이는 2강에서 다루었지만, 여기서는 Qiskit의 메서드, 특히 *합성(synthesis)*이라는 메서드를 활용할 것입니다. 수학적 연산자를 양자 게이트를 사용하는 양자 회로로 합성하는 방법은 여러 가지가 있습니다. 이러한 기법들은 [Qiskit 합성 모듈](https://quantum.cloud.ibm.com/docs/api/qiskit/synthesis)에서 많이 찾아볼 수 있습니다. 우리는 합성을 위해 [`LieTrotter`](https://quantum.cloud.ibm.com/docs/api/qiskit/qiskit.synthesis.LieTrotter) 방식을 사용할 것입니다$\[3\]$(#references) $\[4\]$(#references).

```python
from qiskit.circuit import QuantumRegister
from qiskit.circuit.library import PauliEvolutionGate
from qiskit.synthesis import LieTrotter

evol_gate = PauliEvolutionGate(
    H_op, time=(dt / num_trotter_steps), synthesis=LieTrotter(reps=num_trotter_steps)
)  # `U` operator

qr = QuantumRegister(num_spins)
qc_evol = QuantumCircuit(qr)
qc_evol.append(evol_gate, qargs=qr)

circuits = []
for rep in range(krylov_dim):
    circ = qc_state_prep.copy()

    # Repeating the `U` operator to implement U^0, U^1, U^2, and so on, for power Krylov space
    for _ in range(rep):
        circ.compose(other=qc_evol, inplace=True)

    circ.measure_all()
    circuits.append(circ)
```


이제 회로를 만들었으니 목표 하드웨어에 맞게 최적화할 수 있습니다. 우리는 유틸리티 규모의 QPU를 선택합니다.

```python

from qiskit import generate_preset_pass_manager

```


하드웨어 실행에 최적화된 회로를 만든 후, 이제 목표 하드웨어에서 회로를 실행하고 기저 상태 에너지 추정을 위한 샘플을 수집할 준비가 되었습니다.

## 4. 후처리 결과

다음으로, 크릴로프 차원이 증가함에 따라 누적적으로 개수를 집계합니다. 이 누적 개수를 이용하여 크릴로프 차원이 증가함에 따라 부분 공간을 생성하고 수렴 동작을 분석합니다.

```python
from collections import Counter

counts_cumulative = []
for i in range(krylov_dim):
    counter = Counter()
    for d in counts_all[: i + 1]:
        counter.update(d)

    counts = dict(counter)
    counts_cumulative.append(counts)
```

해밀토니안을 투영하고 대각화하기 위해 [`qiskit-addon-sqd`](https://qiskit.github.io/qiskit-addon-sqd/)의 기능을 사용합니다. 이 애드온은 파울리 스트링 기반 해밀토니안을 부분 공간으로 투영하고 `SciPy`를 사용하여 고유값을 계산하는 기능을 제공합니다.

```python
from qiskit_addon_sqd.counts import counts_to_arrays
from qiskit_addon_sqd.qubit import solve_qubit
```

원칙적으로, 부분 공간을 탐색하기 전에 패턴이 잘못된 비트열을 걸러낼 수 있습니다. 예를 들어, 이 강의에서 다루는 반강자성 해밀턴ian의 바닥 상태는 일반적으로 "위" 스핀과 "아래" 스핀의 수가 같습니다. 즉, 비트열의 "1" 개수는 시스템의 전체 비트(스핀) 수의 정확히 절반이어야 합니다. 다음 함수는 카운트에서 "1"의 개수가 잘못된 비트열을 걸러냅니다.

```python
# Filters out bitstrings that do not have specified number (`num_ones`) of `1` bits.
def postselect_counts(counts, num_ones):
    filtered_counts = {}
    for bitstring, freq in counts.items():
        if bitstring.count("1") == num_ones:
            filtered_counts[bitstring] = freq

    return filtered_counts
```

올바른 개수의 업/다운 전자를 가진 비트스트링을 사용하여 부분 공간을 생성하고 크릴로프 차원이 증가함에 따라 고유값을 계산합니다. 문제의 크기와 사용 가능한 고전적 자원에 따라 부분 공간의 차원을 제어하기 위해 부분 샘플링[SQD 강의 참조](https://quantum.cloud.ibm.com/learning/courses/quantum-diagonalization-algorithms/sqd-implementation)을 적용해야 할 수도 있습니다. 또한, 4강에서 다룬 것과 유사한 구성 복구 개념을 적용할 수 있습니다. 재구성된 고유 상태에서 각 위치의 전자 점유율을 계산하고 이 정보를 사용하여 잘못된 개수의 업/다운 전자를 가진 비트스트링을 수정할 수 있습니다. 이는 관심 있는 독자를 위한 연습 문제로 남겨둡니다.


```python
import numpy as np

num_batches = 10
rand_seed = 0
scipy_kwargs = {"k": 2, "which": "SA"}

ground_state_energies = []
for idx, counts in enumerate(counts_cumulative):
    counts = postselect_counts(counts, num_ones=num_spins // 2)
    bitstring_matrix, probs = counts_to_arrays(counts=counts)

    eigenvals, eigenstates = solve_qubit(
        bitstring_matrix, H_op, verbose=False, **scipy_kwargs
    )
    gs_en = np.min(eigenvals)
    ground_state_energies.append(gs_en)
```

다음으로, 계산된 에너지를 크릴로프 차원의 함수로 나타내고 정확한 에너지와 비교합니다. 정확한 에너지는 고전적인 무차별 대입법을 사용하여 별도로 계산됩니다. 추정된 바닥 상태 에너지가 크릴로프 공간 차원이 증가함에 따라 수렴하는 것을 확인할 수 있습니다. 크릴로프 차원이 5로 제한되지만, 결과는 여전히 인상적인 수렴을 보여주며, 더 큰 크릴로프 차원에서는 수렴이 더욱 개선될 것으로 예상됩니다$\[1\]$(#references).


```python
import matplotlib.pyplot as plt

exact_gs_en = -23.934184
plt.plot(
    range(1, krylov_dim + 1),
    ground_state_energies,
    color="blue",
    linestyle="-.",
    label="estimate",
)
plt.plot(
    range(1, krylov_dim + 1),
    [exact_gs_en] * krylov_dim,
    color="red",
    linestyle="-",
    label="exact",
)
plt.xticks(range(1, krylov_dim + 1), range(1, krylov_dim + 1))
plt.legend()
plt.xlabel("Krylov space dimension")
plt.ylabel("Energy")
plt.ylim([-24, -22.50])
plt.title(
    "Estimating Ground state energy with Sample-based Krylov Quantum Diagonalization"
)
plt.show()
```

### 4.1 이해점검


>  위 플롯에서 수렴을 개선하기 위해 무엇을 할 수 있습니까?"
  
> **답변:**
Krylov 차원을 늘리십시오. 일반적으로 샷 수를 늘릴 수도 있지만, 위 계산에서 이미 상당히 높은 수준입니다.
  

> SKQD가 (a) SQD 및 (b) KQD에 비해 갖는 주요 장점은 무엇입니까?"

> **답변:**
다른 타당한 답변도 있을 수 있지만, 완전한 답변에는 다음 내용이 포함되어야 합니다:
(a) SKQD에는 SQD에는 없는 수렴 보장이 따릅니다. SQD에서는 계산 기저의 바닥 상태 지지와 우수한 중첩을 가지는 변분 추정에 대한 매우 좋은 추측을 하거나, 일련의 변분 추정을 샘플링하기 위해 계산에 변분 성분을 도입해야 합니다.
(b) SKQD는 Hadamard 테스트를 통한 행렬 요소의 비용이 많이 드는 계산을 피하기 때문에 QPU 시간이 훨씬 적게 필요합니다.

## 5. 요약

* 샘플링 크릴로프 기저 상태를 통한 바닥 상태 에너지 추정은 스핀 계열, 응집 물질 문제, 격자 게이지 이론을 포함한 격자 모형에 매우 적합하다. 이 접근법은 VQE에서와 같이 변분 ansatz 내의 많은 매개변수에 대한 최적화를 필요로 하지 않거나, 휴리스틱 ansatz 기반 SQD(예를 들어, 이전 수업의 화학 문제)에서와 같은 최적화를 필요로 하지 않기 때문에 VQE보다 훨씬 더 나은 확장성을 가진다.
  * 회로 깊이를 낮게 유지하려면, 오류 허용 이전 하드웨어에 적합한 격자 문제를 다루는 것이 현명하다.
* SKQD는 VQE에서와 같은 양자 측정 문제를 초래하지 않는다. 추정해야 할 교환 가능한 파울리 연산자들의 그룹이 없다.
* SKQD는 잡음이 있는 샘플에 강건하다. 왜냐하면 문제별 사후 선택 루틴(예를 들어, 문제별 패턴을 따르지 않는 비트스트링을 걸러내는 것)을 사용하거나, 고전적 대각화 오버헤드(즉, 더 큰 부분공간에서 대각화하는 것)를 감수하여 잡음의 효과를 효과적으로 제거할 수 있기 때문이다.
