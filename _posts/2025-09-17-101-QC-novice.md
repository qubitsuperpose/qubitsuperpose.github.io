## 1. Qiskit 개발자 인증 2.0: 준비 방법
- 출처:[Qiskit 개발자 인증 2.0: 준비 방법](https://www.youtube.com/watch?v=FbSvk2krtSk)
 다음과 같이 정리해 드립니다.

### 1.1 **Qiskit Developer Certification 2.0은 무엇인가요?**
IBM Quantum이 Qiskit 개발자 인증을 만들었으며, 최근 **버전 2.0이 출시**되었습니다. 이 인증은 Qiskit을 사용하여 양자 컴퓨팅 분야에서 경력을 쌓고자 하는 사람들을 위해 고안되었습니다. 이 시험에 합격하면 이력서에 추가할 수 있는 강력한 자격 증명과 LinkedIn 프로필에 사용할 수 있는 배지를 얻게 되며, 이는 채용 관리자에게 눈에 띄는 요소가 될 수 있습니다.

### 1.2 **시험 정보:**
*   **시험 시간**: 90분.
*   **문제 수**: 68개의 객관식 문제(47개의 문제를 맞추어야, 70점이상 획득)
*   **시험 내용**: Qiskit을 사용하고 이해하는 능력을 테스트합니다.
*   **사용 제한 자료**: 시험 중에는 인터넷, 계산기, 종이 및 연필과 같은 자료에 접근할 수 없습니다.

### 1.3 **시험 준비를 위한 필수 배경 지식:**
성공적인 시험 응시를 위해 다음 지식에 익숙하고 편안해야 합니다:
*   **선형 대수학**
*   **단위원 상의 삼각법**
*   **양자 역학의 기초**
    *   이 지식이 부족하다면, **John Watrous의 "Understanding Quantum Information and Computation" 시리즈의 1장과 2장**부터 시작하는 것이 좋은 기초가 될 수 있습니다.

### 1.4 **시험에 포함되는 주요 영역:**
시험은 다음 주요 영역을 다룹니다:
*   단일 큐비트 게이트 및 연산
*   양자 회로 구성 및 시각화
*   매개변수화된 회로
*   Qiskit Runtime 및 Sampler v2 및 Estimator v2를 포함한 프리미티브
*   오류 완화 및 억제
*   Qiskit Runtime Job 객체
*   OpenQASM 3
*   Qiskit Runtime REST API

### 1.5 **권장 학습 자료 및 방법:**
*   **Qiskit 문서**: 강점이 있는 주제든 약점이 있는 주제든 **Qiskit 문서는 가장 좋은 친구**입니다. 예제와 API 레퍼런스를 깊이 탐구하는 것이 중요합니다.
*   **Qiskit YouTube 채널 및 IBM Quantum Learning 플랫폼**:
    *   "Understanding Quantum Information and Computation" 시리즈
    *   "Quantum Computing in Practice" 시리즈
*   **실습 경험**: Python을 사용하여 **Qiskit을 직접 다루는 실습**은 시험에 필요한 실험적 지식을 제공할 것입니다. 나열된 주제에 집중하고 충분한 시간을 들여 현명하게 학습하는 것이 중요합니다.

## 2. 학습 가이드 Qiskit 2.0 Certification

### <span style="color:yellow"> 1. Perform Quantum operations
* 1.1 Define Pauli Operations 
* 1.2 Apply quantum operations

### <span style="color:yellow"> 2. Visualize quantum circuits, measurements, and states
* 2.1 Visualize quantum circuits
* 2.2 Visualize quantum measurements
* 2.3 Visualize quantum states

### <span style="color:yellow"> 3. Create quantum circuits
* 3.1 Construct dynamic circuits
* 3.2 Construct parameterized circuits
* 3.3 Transpile and Optimize circuits
* 3.4 Construct basic quantum circuits

### <span style="color:yellow"> 4. Run quantum circuits
* 4.1 Demonstrate and understanding of execution modes such as: session with dedicated, priority, and batch mode
    * Sessions
    * Batch
* 4.2 Demonstrate understanding of how to run quantum circuits with real hardware using Qiskit Runtime primitives and applying broadcasting rules
    * Primitives
    * Broadcasting rules
    * Input/Output
    * Jobs

### <span style="color:yellow"> 5. Use the sampler primitive
* 5.1 Set sampler primitive options such as dynamical decoupling
* 5.2 Understand the theoretical background behind the sampler primitive

### <span style="color:yellow"> 6. Use the estimate primitive
* 6.1 Set estimator primitive options(such as resilience levels)
* 6.2 Understand the theoretical background behind the estimator primitive

### <span style="color:yellow"> 7. Retrieve and analyze the result of quantum circuits
* 7.1 Retrieve previous experiment results (session/Qiskit Runtime)
* 7.2 Monitor jobs

### <span style="color:yellow"> 8. Operate with OpenQASM
* 8.1 Structure types in OpenQASM3 programs
* 8.2 Interpret OpenQASM semantics
* 8.3 Interoperate different versions of OpenQASM with Qiskit
* 8.4 Interact with the Qiskit IBM Runtime REST API
