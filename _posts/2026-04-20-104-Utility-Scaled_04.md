## 1. 그로버 알고리즘(Grover's algorithm) 소개

이 노트북은 **양자 컴퓨팅의 실용성으로 가는 길(Path to Utility in Quantum Computing)** 강의 시리즈의 네 번째 노트북입니다. 이 노트북에서는 **그로버 알고리즘(Grover's algorithm)** 에 대해 학습합니다.

그로버 알고리즘은 고전적인 탐색 방법에 비해 **이차(quadratic)*속도 향상**을 제공하는 것으로 잘 알려진 대표적인 양자 알고리즘 중 하나입니다. 고전 컴퓨팅에서는 $N$개의 항목으로 이루어진 **정렬되지 않은 데이터베이스(unsorted database)** 를 탐색하는 데 $O(N)$의 시간 복잡도가 필요합니다. 이는 최악의 경우 모든 항목을 하나씩 검사해야 할 수도 있음을 의미합니다. 반면, 그로버 알고리즘은 양자역학의 원리를 활용하여 목표 항목을 더욱 효율적으로 찾아냄으로써 이러한 탐색을 $O(\sqrt{N})$의 시간 복잡도로 수행할 수 있습니다.

이 알고리즘은 **진폭 증폭(amplitude amplification)** 이라는 과정을 사용합니다. 이는 양자 중첩(quantum superposition) 상태에서 정답 상태(correct answer state*의 **확률 진폭(probability amplitude)** 을 증가시켜, 해당 상태가 더 높은 확률로 측정될 수 있도록 하는 과정입니다. 이러한 속도 향상 덕분에 그로버 알고리즘은 단순한 데이터베이스 탐색을 넘어 다양한 응용 분야에서 유용하게 활용되며, 특히 데이터 집합의 크기가 큰 경우 더욱 큰 장점을 제공합니다. 알고리즘에 대한 자세한 설명은 [Grover's algorithm notebook](/learning/courses/fundamentals-of-quantum-algorithms/grover-algorithm/introduction)에서 확인할 수 있습니다.

### 1.1 그로버 알고리즘의 기본 구조

그로버 알고리즘은 다음의 네 가지 주요 구성 요소로 이루어집니다.

1. **초기화(Initialization)**:*가능한 모든 상태에 대한 중첩(superposition)을 생성합니다.
2. **오라클(Oracle)**:*목표 상태(target state)의 위상(phase)을 반전시켜 해당 상태를 표시하는 오라클 함수를 적용합니다.
3. **확산 연산자(Diffusion Operator)**: 표시된 상태(marked state)의 확률을 증폭하기 위해 일련의 연산을 적용합니다.

이러한 각 단계는 알고리즘이 효율적으로 동작하는 데 중요한 역할을 합니다. 각 단계에 대한 자세한 설명은 이후에 제공합니다.

4개의 요소로 이루어진 리스트를 생각해 봅시다. 우리의 목표는 특정 조건을 만족하는 요소의 인덱스를 찾는 것입니다. 예를 들어, 값이 2인 요소의 인덱스를 찾고자 합니다. 이 예에서 양자 상태 $\|01\rangle$은 값 2가 위치한 곳을 가리키므로, 해당 조건을 만족하는 요소의 인덱스를 나타냅니다.

### 2단계: 대상 하드웨어에 맞게 최적화

#### 1: 초기화

초기화 단계에서는 가능한 모든 상태의 중첩을 생성합니다. 이는 n개의 큐비트를 가진 레지스터의 각 큐비트에 하드마르 게이트를 적용하여 $2^n$개의 상태가 균등하게 중첩되도록 함으로써 이루어집니다. 수학적으로는 다음과 같이 표현할 수 있습니다.
<p>
$$
\frac{1}{\sqrt{N}} \sum_{x=0}^{N-1} |x\rangle
$$
</p>

여기서 $N = 2^n$은 가능한 모든 상태의 총 개수입니다. 또한, 보조 큐비트의 상태를 $\|-\rangle$로 변경합니다.

```python
def initialization(circuit):
    # Initialization
    n = circuit.num_qubits
    # For input qubits
    for qubit in range(n - 1):
        circuit.h(qubit)
    # For the ancilla bit
    circuit.x(n - 1)
    circuit.h(n - 1)
    circuit.barrier()
    return circuit
```

```python
n = 2
qr = QuantumRegister(n, "q")
anc = QuantumRegister(1, "ancillary")
initialization_circuit = QuantumCircuit(qr, anc)

initialization(initialization_circuit)
initialization_circuit.draw(output="mpl", idle_wires=False)
```

#### 2: 오라클

오라클은 그로버 알고리즘의 핵심 요소입니다. 오라클은 위상 변화를 적용하여 목표 상태를 표시하며, 일반적으로 해당 상태와 연관된 진폭의 부호를 반전시키는 방식을 취합니다. 오라클은 대체로 문제마다 고유하게 설계되며, 목표 상태를 식별하는 기준에 기반하여 구축됩니다. 수학적으로 오라클은 다음과 같은 변환을 적용합니다:
<p>
$f(x) =
\begin{cases}
1, & \text{if } x = x_{\text{target}} \\
0, & \text{otherwise}
\end{cases}$
</p>

이러한 위상 반전은 위상 킥백을 통해 목표 상태의 진폭에 음의 부호를 적용함으로써 이루어집니다.


```python
def oracle(circuit):
    circuit.x(1)
    circuit.ccx(0, 1, 2)
    circuit.x(1)
    circuit.barrier()
```

```python
n = 2
qr = QuantumRegister(n, "q")
anc = QuantumRegister(1, "ancillary")
oracle_circuit = QuantumCircuit(qr, anc)

oracle(oracle_circuit)
oracle_circuit.draw(output="mpl", idle_wires=False)
```

#### 3: 확산 연산자

진폭 증폭 과정이야말로 그로버 알고리즘을 고전적 탐색과 구별 짓는 요소입니다. 오라클이 목표 상태를 표시한 후, 우리는 이 표시된 상태의 진폭을 증가시키는 일련의 연산을 적용하여 측정 시 해당 상태가 관측될 확률을 높입니다. 이 과정은 효과적으로 평균 진폭을 기준으로 한 반전을 수행하는 **확산 연산자**를 통해 이루어집니다. 수학적 연산은 다음과 같습니다:
<p>
$D = 2|\psi\rangle\langle\psi| - I$
</p>

여기서 $D$는 확산 연산자, $I$는 항등 행렬, 그리고 $\|\psi\rangle$는 균등 중첩 상태입니다. 표시된 상태가 측정될 확률을 최대화하기 위해 오라클과 확산 연산자의 조합이 대략 $\sqrt{N}$번 적용됩니다.

```python
def diffusion(circuit):
    input_qubits = circuit.num_qubits - 1
    circuit.h(range(0, input_qubits))
    circuit.x(range(0, input_qubits))
    circuit.h(input_qubits - 1)
    circuit.mcx([i for i in range(0, input_qubits - 1)], input_qubits - 1)
    circuit.h(input_qubits - 1)
    circuit.x(range(0, input_qubits))
    circuit.h(range(0, input_qubits))
    circuit.barrier()
```

```python
n = 2
qr = QuantumRegister(n, "q")
anc = QuantumRegister(1, "ancillary")
diffusion_circuit = QuantumCircuit(qr, anc)

diffusion(diffusion_circuit)
diffusion_circuit.draw(output="mpl", idle_wires=False)
```

### 2.2 2큐비트 그로버 탐색 예시.

```python
n = 2
qr = QuantumRegister(n, "q")
anc = QuantumRegister(1, "ancillary")
meas = ClassicalRegister(3, "meas")
grover_circuit = QuantumCircuit(qr, anc, meas)
```

```python
# the number of iterations
num_iterations = 1
```

```python
# Let's do Grover search
initialization(grover_circuit)

for i in range(0, num_iterations):
    oracle(grover_circuit)
    diffusion(grover_circuit)

# Clear the ancilla bit
grover_circuit.h(n)
grover_circuit.x(n)
grover_circuit.measure_all(add_bits=False)

grover_circuit.draw(output="mpl", idle_wires=False)
```
