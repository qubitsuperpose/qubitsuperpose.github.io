## 1. 서론

어떤 유틸리티 규모의 양자 문제를 해결하기 위해서는, 양자 컴퓨터에서 정보를 한 큐비트에서 다른 큐비트로 옮겨야 할 것이다. 이를 위한 잘 알려진 프로토콜들이 있지만, 가장 기초적인 것들 중 일부는 원격의 당사자들 사이에 정보를 보내는 맥락에서 제시되었다. 이 강의 전체에 걸쳐, 우리는 때때로 "원격의 친구들이 정보를 보내는 것"과 같은 이 맥락에 부합하는 표현을 사용할 것이다. 그러나 이러한 프로토콜들이 양자 컴퓨팅에서 더 넓은 의의를 가진다는 점을 명심하기 바란다. 이 강의에서 우리는 다음과 같은 양자 통신 프로토콜들을 고려한다:

* **양자 텔레포테이션**
  공유된 얽힌 상태(때때로 e-bit라고 불림)를 사용하여 알려지지 않은 양자 상태를 원격의 친구에게 보내며, 추가적인 고전적 통신을 필요로 한다.
* **양자 초밀도 코딩**
  단일 큐비트를 원격의 친구에게 보냄으로써 두 비트의 정보를 보내는 방법(다시 사전에 공유된 얽힌 큐비트들을 사용한다).

이러한 주제들과 관련된 더 많은 배경 지식을 위해, 우리는 [Entanglement in action](/learning/courses/basics-of-quantum-information/entanglement-in-action/introduction)에 관한 양자 정보 기초 강의 4를 추천한다.

위의 설명에서 "알려지지 않은 양자 상태"는 단순히 이전 강의에서 설명된 형태의 상태를 가리킨다:
<p>
$$
|\psi\rangle =\alpha|0\rangle+\beta|1\rangle
$$
</p>

여기서 $\alpha$와 $\beta$는 $\|\alpha\|^2+\|\beta\|^2 = 1$을 만족하는 복소수들이다. 이를 통해 우리는 양자 상태를 다음과 같이 쓸 수 있다:

<p>
$$
|\psi\rangle =\cos\frac{\theta}{2}|0\rangle+e^{i\varphi}\sin\frac{\theta}{2}|1\rangle=
\left(
\begin{matrix}
\cos\frac{\theta}{2}\\
e^{i\varphi}\sin\frac{\theta}{2}
\end{matrix}
\right)
$$
</p>

우리는 임의의 양자 상태에 있는 정보를 전송할 수 있기를 원하기 때문에, 그러한 상태를 생성하는 것이 이 강의를 시작하는 지점이 될 것이다.

## 2. 밀도 행렬

우리는 양자 상태 $\|\psi \rangle$를 밀도 행렬(density matrix)로도 표현할 수 있습니다. 이 형태는 순수 양자 상태들의 확률적 혼합(probabilistic mixture)을 나타내는 데 유용합니다. 단일 큐비트의 경우, 다음과 같이 쓸 수 있습니다.

<p>
$$
|\psi \rangle \langle \psi| \equiv  \rho = \left(
\begin{pmatrix}
\cos\frac{\theta}{2}\\
e^{i\varphi}\sin\frac{\theta}{2}
\end{pmatrix}
\right)
\left(
\begin{pmatrix}
\cos\frac{\theta}{2} & e^{-i\varphi}\sin\frac{\theta}{2}
\end{pmatrix}
\right)
=\frac{1}{2}\left(\begin{pmatrix}
1+\cos\theta & e^{-i\varphi}\sin\theta\\
e^{-i\varphi}\sin\theta & 1-\cos\theta
\end{pmatrix}\right)
$$
</p>

밀도 행렬 $\rho$은 아래와 같이 파울리 행렬들의 선형 합으로 표현된다는 점에 주목하세요.

<p>
$$
\rho = \frac{1}{2}\bigl( \textbf{I} + (\sin{\theta}\cos{\varphi})\textbf{X}+  (\sin{\theta}\sin{\varphi})\textbf{Y} +  (\cos{\theta})\textbf{Z} \bigr)
$$
</p>

또는 일반적으로,

<p>
$$
\rho = \frac{1}{2}(\textbf{I} + r_{x}\textbf{X}+ r_{y}\textbf{Y} + r_{z}\textbf{Z})
$$
</p>

여기서
$r_{x}^2+r_{y}^2+r_{z}^2=1$입니다.

그리고 Bloch 벡터는 $\textbf{r} = (r_{x}, r_{y}, r_{z})$입니다.

이제 무작위 숫자를 사용하여 임의의 양자 상태를 만들어 봅시다.

```python
import numpy as np

# create a random 1-qubit state from a random (theta, varphi) to define r vector
np.random.seed(1)  # fixing seed for repeatibility

theta = np.random.uniform(0.0, 1.0) * np.pi  # from 0 to pi
varphi = np.random.uniform(0.0, 2.0) * np.pi  # from 0 to 2*pi


def get_r_vec(theta, varphi):
    rx = np.sin(theta) * np.cos(varphi)
    ry = np.sin(theta) * np.sin(varphi)
    rz = np.cos(theta)
    return (rx, ry, rz)


# get r vector
rx, ry, rz = get_r_vec(theta, varphi)

print("theta=" + str(theta), ",varphi=" + str(varphi))
print("(rx, ry, rz) = (" + str(rx) + ", " + str(ry) + ", " + str(rz) + ")")
```

```
theta=1.3101132663588946 ,varphi=4.525932273597346
(rx, ry, rz) = (-0.1791150283307452, -0.9494670044331133, 0.2577405946274022)
```

이 블로흐 벡터를 블로흐 구면에 나타낼 수 있습니다.

```python
from qiskit.visualization import plot_bloch_vector

r = [rx, ry, rz]
plot_bloch_vector(r)
```

## 3. 양자 상태 토모그래피

계산 기저($\|0 \rangle$와 $\|1 \rangle$)로만 양자 상태를 측정하면 위상 정보(복소수 정보)가 손실됩니다. 하지만 준비 과정을 반복하여 $\|\psi \rangle$의 많은 복사본을 얻을 수 있다면(상태를 복제할 수는 없지만 준비 과정은 반복할 수 있습니다), 밀도 행렬 $\rho$에 대해 **양자 상태 토모그래피(quantum state tomography)**를 수행하여 $r_{x}, r_{y}, r_{z}$ 값을 추정할 수 있습니다. 주어진 형태는 다음과 같습니다:

<p>
$$
\rho = \frac{1}{2}(\textbf{I} + r_{x}\textbf{X}+ r_{y}\textbf{Y} + r_{z}\textbf{Z})
$$
</p>

이때 다음이 성립합니다:

<p>
$$
Tr(\textbf{X} \rho) = r_{x}, \quad Tr(\textbf{Y} \rho) = r_{y}, \quad Tr(\textbf{Z} \rho) = r_{z}
$$
</p>

$r_{z}$의 경우,
<p>
$$
Tr(\textbf{Z} \rho) = \langle 0|\textbf{Z} \rho|0 \rangle + \langle 1|\textbf{Z} \rho|1 \rangle
$$

$$
= \langle 0|(|0 \rangle\langle 0|-|1 \rangle\langle 1|) \rho|0 \rangle +\langle 1|(|0 \rangle\langle 0|-|1 \rangle\langle 1|) \rho|1 \rangle
$$

$$
=\langle 0|\rho|0 \rangle- \langle 1| \rho|1 \rangle
$$

$$
=\langle 0|\psi\rangle\langle \psi|0 \rangle - \langle 1| \psi\rangle\langle \psi|1 \rangle
$$

$$
=|\alpha|^2-|\beta|^2
$$
</p>

마지막 변환은 $\|\psi \rangle =\alpha\|0\rangle+\beta\|1\rangle$인 경우에 해당합니다. 따라서 $\|0 \rangle$의 확률 − $\|1 \rangle$의 확률을 통해 $r_{z}$를 얻을 수 있습니다.

### 3.1 $r_z$ 값 추정하기
$r_z$를 추정하기 위해 양자 상태를 생성한 후 측정합니다. 그런 다음 준비와 측정을 여러 번 반복합니다. 마지막으로 측정 통계를 사용하여 위의 확률들을 추정하고, 이를 통해 $r_z$를 추정합니다.

무작위 양자 상태를 생성하기 위해 매개변수 $\theta, \varphi$를 가진 일반 유니터리 게이트 $U$를 사용합니다. (자세한 내용은 [U-gate](https://quantum.cloud.ibm.com/docs/api/qiskit/qiskit.circuit.library.UGate)를 참조하세요.)


```python
from qiskit import QuantumCircuit

# create a 1-qubit quantum state psi from theta, varphi parameters
qc = QuantumCircuit(1, 1)
qc.u(theta, varphi, 0.0, 0)

# measure in computational basis
qc.measure(0, 0)

qc.draw(output="mpl")
```

![](https://quantum.cloud.ibm.com/assets-docs-learning/_next/image?url=%2Flearning%2Fimages%2Fcourses%2Futility-scale-quantum-computing%2Fteleportation%2Fextracted-outputs%2Fb1327267-f2fe-4a61-91d9-328c672b557e-0.avif&w=640&q=75)

AerSimulator`를 사용하여 계산 기반에서 이를 측정하고 $r_z$를 추정할 것입니다.

```python
# see if the expected value of measuring in the computational basis
# approaches the limit of rz
from qiskit_aer import AerSimulator
from qiskit.transpiler.preset_passmanagers import generate_preset_pass_manager
from qiskit_ibm_runtime import Sampler
from qiskit.visualization import plot_histogram

# Define backend
backend = AerSimulator()
nshots = 1000  # or 10000
# nshots = 10000

# Transpile to backend
pm = generate_preset_pass_manager(backend=backend, optimization_level=1)
isa_qc = pm.run(qc)

# Run the job
sampler = Sampler(mode=backend)
job = sampler.run([isa_qc], shots=nshots)
result = job.result()

# Extract counts data
counts = result[0].data.c.get_counts()
print(counts)

# Plot the counts in a histogram

plot_histogram(counts)
```

```
{'1': 375, '0': 625}
```

양자 상태 단층 촬영법을 사용하여 $r_z$ 값을 추정했습니다. 이 경우 "랜덤" 상태에 대한 매개변수를 선택했기 때문에 $r_z$ 값을 알고 있으며, 우리의 결과를 검증할 수 있습니다. 하지만 실용적인 규모의 연구에서는 검증이 항상 이렇게 간단하지는 않습니다. 양자 결과 검증에 대해서는 이 강의 후반부에서 더 자세히 다루겠습니다. 지금은 우리의 추정치가 상당히 정확했다는 점만 기억해 두십시오.

## 4. 양자 순간이동 (Quantum Teleportation)

앨리스(Alice)라는 인물이 멀리 떨어져 있는 친구 밥(Bob)에게 미지의 양자 상태 $\|\psi \rangle$를 보내고 싶어하는 상황을 생각해 봅시다. 그들은 오직 고전적 통신(예: 이메일이나 전화)만 사용할 수 있다고 가정합니다. 앨리스는 양자 상태를 복제할 수 없습니다(노클로닝 정리 때문). 만약 동일한 준비 과정을 여러 번 반복할 수 있다면, 앞서 했던 것처럼 통계를 쌓을 수 있을 것입니다. 하지만 단 하나의 미지의 상태만 있다면 어떻게 될까요? 이 상태는 여러분이 연구하고 싶은 물리적 과정에서 나왔을 수도 있고, 더 큰 양자 계산의 일부일 수도 있습니다. 이 경우 앨리스는 어떻게 그 상태를 밥에게 보낼 수 있을까요?

그녀는 가능합니다. 앨리스와 밥이 **공유된 얽힌 상태**라는 귀중한 양자 자원을 가지고 있다면 말입니다. 이전 강의에서 소개된 벨 상태(Bell state)와 같은 것입니다:  
<p>
$$\frac{|00\rangle + |11\rangle}{\sqrt{2}}.$$</p>

이것을 때때로 “EPR 쌍” 또는 “e-bit”(얽힘의 기본 단위)이라고 부르기도 합니다. 앨리스가 이런 얽힌 상태를 밥과 공유하고 있다면, 그녀는 일련의 양자 연산을 수행하고 **두 비트의 고전적 정보**를 그에게 보냄으로써 미지의 양자 상태를 밥에게 *순간이동(teleport)*할 수 있습니다.

### 4.1 양자 순간이동 프로토콜

**가정**: 앨리스는 보내려는 미지의 양자 상태 $\|\psi \rangle$를 가지고 있습니다. 앨리스와 밥은 2-큐비트 얽힌 상태(또는 e-bit)를 공유하고 있으며, 각자 그중 하나의 큐비트를 물리적으로 가지고 있습니다.

여기서는 설명 없이 절차를 먼저 개요로 제시합니다. 자세한 구현은 아래에서 다루겠습니다.

1. 앨리스는 CNOT 게이트를 사용하여 $\|\psi \rangle$를 자신의 e-bit 부분과 얽힙니다.
2. 앨리스는 $\|\psi \rangle$에 Hadamard 게이트를 적용한 후, 자신의 두 큐비트를 모두 계산 기저(computational basis)에서 측정합니다.
3. 앨리스는 자신의 측정 결과(“00”, “01”, “10”, 또는 “11”)를 밥에게 보냅니다.
4. 밥은 앨리스의 두 비트 정보에 따라 자신의 e-bit 부분에 *보정(correction)* 연산자를 수행합니다.
   * “00”인 경우: 아무것도 하지 않음
   * “01”인 경우: X 게이트 적용
   * “10”인 경우: Z 게이트 적용
   * “11”인 경우: iY = ZX 게이트 적용
5. 밥의 e-bit 부분이 $\|\psi \rangle$가 됩니다.

이 내용은 [Basics of Quantum Information](https://quantum.cloud.ibm.com//learning/courses/basics-of-quantum-information/entanglement-in-action/introduction)에서도 더 자세히 다루어집니다. 하지만 Qiskit으로 직접 구현해 보면 상황이 더욱 명확해질 것입니다.

### 4.3 실행하고 U inverse를 적용하여 결과 확인하기

위에서 우리는 텔레포트된 상태가 올바르게 보이는 것을 시각적으로 확인했습니다. 양자 상태가 올바르게 텔레포트되었는지 확인하는 또 다른 방법은 Bob의 큐비트에 $U$ 게이트의 역(inverse)을 적용하여 '0'을 측정할 수 있게 하는 것입니다. 즉, $U^{-1}U$가 항등(identity)이기 때문에, Bob의 큐비트가 $U\|0\rangle$으로부터 생성된 상태에 있다면, 역을 적용하면 $U^{-1}U\|0\rangle=\|0\rangle$이 되어야 합니다.

```python
# Apply the inverse of u-gate to measure |0>
qc.u(theta, varphi, 0.0, 2).inverse()  # inverse of u(theta,varphi,0.0)
qc.measure(2, 2)  # add measurement gate

qc.draw(output="mpl")
```

```python
real_counts = result_real[0].data.c.get_counts()
bobs_counts = marginal_counts(real_counts, [bobs_qubit])
plot_histogram(bobs_counts)
```

### 4.5 양자 텔레포테이션의 핵심 요점

얽힌 큐비트 쌍(e-비트)을 공유함으로써 양자 상태를 멀리 떨어진 친구에게 전송할 수 있습니다.

1. 양자 텔레포테이션으로 양자 상태를 빛보다 빠르게 보낼 수 있을까요?
아니요, 앨리스는 밥에게 측정 결과를 고전적인 방식으로 알려줘야 하기 때문입니다.

2. 양자 텔레포테이션은 양자 상태 복제를 금지하는 "복제 금지 정리"를 위반할까요?
아니요, 앨리스에게 전달된 큐비트 중 하나에 저장된 원래 양자 상태는 측정 과정에서 손실되기 때문입니다. 해당 상태는 $\|0\rangle$ 또는 $\|1\rangle$로 붕괴됩니다.



## 5. 초고밀도 코딩

거의 동일한 설정을 다른 목적으로 사용할 수 있습니다. 앨리스가 밥에게 두 비트의 고전 정보를 보내고 싶지만, 밥과 고전적 통신을 할 수 있는 수단이 없다고 가정해 봅시다. 하지만 앨리스는 밥과 얽힌 큐비트 쌍을 공유하고 있으며, 자신의 큐비트를 밥의 위치로 보낼 수 있습니다. 양자 텔레포테이션 프로토콜과의 차이점을 주목해 보세요. 텔레포테이션에서는 두 친구 사이에 고전적 통신이 가능했고, 목표는 양자 상태를 보내는 것이었습니다. 하지만 여기서는 고전적 통신이 불가능하므로 큐비트 전송을 통해 두 비트의 고전 정보를 공유합니다.



### 5.1 초밀도 코딩 프로토콜

**가정**: 앨리스는 두 비트의 정보 $a_1a_2 \in \{00, 01, 10, 11\}$를 가지고 있다. 앨리스와 밥은 얽힌 쌍(e-bit)을 공유하고 있지만, 고전적인 통신은 할 수 없다.

1. 앨리스는 e-bit 중 자신의 부분에 대해 다음 연산 중 하나를 수행한다.
   * $a_1a_2 = 00$인 경우, 아무것도 하지 않는다.
   * $a_1a_2 = 01$인 경우, Z 게이트를 적용한다.
   * $a_1a_2 = 10$인 경우, X 게이트를 적용한다.
   * $a_1a_2 = 11$인 경우, Z 게이트와 X 게이트를 적용한다.
2. 앨리스는 e-bit 중 자신의 부분을 밥의 위치로 보낸다.
3. 밥은 앨리스로부터 받은 큐비트를 제어 큐비트로, 자신의 큐비트를 타겟 큐비트로 하는 CNOT 게이트를 적용한 후, 앨리스로부터 받은 큐비트에 H 게이트를 적용하고 두 큐비트를 측정한다. 가능한 초기 상태와 밥의 연산 결과는 다음과 같다:
<p>
$$
\frac {|00\rangle + |11\rangle}{\sqrt 2} \rightarrow CX_{01}\otimes H_0 \rightarrow |00\rangle
$$

$$
\frac {|00\rangle - |11\rangle}{\sqrt 2} \rightarrow CX_{01}\otimes H_0 \rightarrow |01\rangle
$$

$$
\frac {|10\rangle + |01\rangle}{\sqrt 2} \rightarrow CX_{01}\otimes H_0 \rightarrow |10\rangle
$$

$$
\frac {|10\rangle - |01\rangle}{\sqrt 2} \rightarrow CX_{01}\otimes H_0 \rightarrow -|11\rangle
$$
</p>

참고: $-\|11\rangle$의 음의 부호는 전역 위상(global phase)이므로 측정할 수 없다.

### 5.2 5.2 초고밀도 코딩을 모방한 양자 회로

초고밀도 코딩 프로토콜을 기반으로 아래와 같이 초고밀도 코딩 회로를 구성할 수 있습니다. 앨리스가 밥에게 전달하려는 메시지 `msg`를 바꿔가며 시도해 보세요.

```python
from qiskit import QuantumCircuit
```

Qiskit 패턴 단계는 코드 주석에 명시되어 있습니다.

```python
# Step 1: Map problem to quantum circuits and operators
# Create 2-qubits circuit
qc = QuantumCircuit(2, 2)

# Eve creates EPR pair and send q0 to Alice and q1 to Bob
qc.h(0)
qc.cx(0, 1)
qc.barrier()

# set message which Alice wants to transform to Bob
msg = "11"  # You can change the message

if msg == "00":
    pass
elif msg == "10":
    qc.x(0)
elif msg == "01":
    qc.z(0)
elif msg == "11":
    qc.z(0)
    qc.x(0)

qc.barrier()
# Bob receives EPR qubit from Alice and performs unitary operations
qc.cx(0, 1)
qc.h(0)
qc.barrier()

# Bob measures q0 and q1
qc.measure(0, 0)
qc.measure(1, 1)

qc.draw(output="mpl")
```
