# 2. Visualize Quantum Circuits


## 1. 양자 회로 시각화

목표: 양자 회로 시각화

**개요:** 이 노트북은 qiskit의 내장 시각화 도구를 사용하여 양자 회로를 시각화하는 다양한 방법을 다룹니다.
적절한 시각화는 회로 구조 이해, 디버깅, 그리고 양자 알고리즘 제시에 도움이 됩니다.

```python
# Setup: Import necessary libraries
import numpy as np
from qiskit import QuantumCircuit,QuantumRegister, ClassicalRegister

print("Libraries imported successfully.")
```

### 1. 기본 회로 생성 및 텍스트 시각화

간단한 양자 회로를 만들고 텍스트 형식으로 시각화하는 것부터 시작해 보겠습니다.

```python
# Build a quantum circuit with 3 quantum bits and 3 classical bits
circuit = QuantumCircuit(3, 3)

# Add quantum gates
circuit.h(0)                    # Hadamard gate on qubit 0
circuit.cx(0, 1)                # CNOT gate with control=0, target=1
circuit.cx(0, 2)                # CNOT gate with control=0, target=2
                
circuit.measure(range(3), range(3))  # Measure all qubits to classical bits

print("Text representation of the circuit:")
print(circuit)
```


### 2. 기본 그리기(텍스트 기반)

매개변수 없이 `draw()` 메서드를 사용하면 동일한 텍스트 형식으로 회로를 나타냅니다. 이는 터미널 환경에서 빠른 디버깅에 유용합니다.

```python
# Default text-based circuit drawing
print("Text representation:")
circuit.draw()
```

### 3. 다른 시각화

출판 품질의 회로도를 위해 두 가지 대안이 있습니다. 하나는 `mpl` (Matplotlib) 출력입니다. 이것은 적절한 간격과 스타일링으로 깔끔하고 전문적인 시각화를 제공합니다. 다른 하나는 `latex` LaTex 출력이지만 별도의 패키지 설치가 필요합니다.

```python
# High-quality visualization using matplotlib
print("Matplotlib visualization:")
circuit.draw(output="mpl")
```


#### 1. 출력 저장하기

출력 이미지는 파일에 직접 저장하고 나중에 이미지 뷰어에서 볼 수 있습니다.

```python
#Save image
circuit.draw(output="mpl", filename="circuit.jpeg")
```

### 4. 더 복잡한 회로

양자 회로는 여러 개의 양자 및 고전 레지스터를 가질 수 있습니다. 이 예제는 다음을 보여줍니다:
- 더 나은 구성을 위한 명명된 레지스터
- 회로 섹션을 구분하는 장벽 연산
- 다른 레지스터 간의 측정

```python
# Create named quantum and classical registers
qreg_q = QuantumRegister(7, 'q')    # Quantum register 'qreq_q' with 7 qubits
creg_c = ClassicalRegister(7, 'c')  # Classical register for measurements from 'creg_c'
 
# Build a more complex circuit
circuit = QuantumCircuit(qreg_q, creg_c)


circuit.h(qreg_q[0])                # Hadamard on qubit 0
circuit.cx(qreg_q[0], qreg_q[1])    # CNOT on qubits 0 and 1
 
# Barrier
circuit.barrier()
 
circuit.cx(qreg_q[0], qreg_q[2])    # CNOT on qubits 0 and 2
circuit.cx(qreg_q[1], qreg_q[3])    # CNOT on qubits 1 and 3

circuit.cx(qreg_q[0], qreg_q[4])    # CNOT on qubits 0 and 4
circuit.cx(qreg_q[1], qreg_q[5])    # CNOT on qubits 1 and 5

circuit.cx(qreg_q[0], qreg_q[6])    # CNOT on qubits 0 and 6

circuit.barrier()
# Measurement phase
circuit.measure(qreg_q, creg_c)     # Measure quantum register 'qreg_q' to classical 'creg_c'


# Draw the complete circuit
print("circuit with multiple barriers:")
circuit.draw(output="mpl")
```


### 5. 비트 순서 반전

기본적으로 Qiskit은 리틀 엔디언 표기법(qubit 0이 상단)을 사용합니다. `reverse_bits` 옵션은 다른 규칙이나 개인 선호도에 맞게 표시 순서를 변경합니다.

```python
# Draw with reversed bit order (big-endian style)
print("Circuit with reversed bit order (qubit 0 at the bottom of the diagram):")
circuit.draw(output="mpl", reverse_bits=True)
```


### 6. 장벽 숨기기

장벽은 회로 컴파일 및 최적화에 유용하지만 프레젠테이션이나 출판물에서 더 깔끔한 시각화를 위해 숨길 수 있습니다.

```python
# Draw without showing barrier lines
print("Circuit visualization with barriers hidden:")
circuit.draw(output="mpl", plot_barriers=False)
```

### 7. 접기 기능이 있는 텍스트 출력

게이트가 많은 회로의 경우 텍스트 출력이 매우 넓어질 수 있습니다. `fold` 매개변수는 가독성을 위해 줄 길이를 제한합니다.

```python
# Display with line folding at 50 characters
print("Text output folded at 50 characters:")
circuit.draw(output="text", fold=50)
```

### 8. 스타일링

`draw()` 메서드에서 `style` 매개변수를 사용하여 회로의 모양을 변경할 수 있습니다. 지원되는 스타일 이름은 "iqp", "iqp-dark", "clifford", "textbook" 및 "bw"이며, json 파일을 사용할 수 있지만 시각화 사양을 완전히 지정해야 합니다.

파일은 qiskit/visualization/circuit/styles, 현재 작업 디렉토리, 그리고 ~/.qiskit/settings.conf에 지정된 위치에서 검색됩니다.

```python
print("Circuit with black and white styling:")
circuit.draw(output="mpl", style="bw")
```

Qiskit은 스타일 딕셔너리를 통해 회로 모양의 광범위한 사용자 지정을 허용합니다. 이는 프레젠테이션, 교육 자료 또는 출판 스타일에 맞추는 데 유용합니다.

```python
# Custom style dictionary for circuit appearance
style = {
    "backgroundcolor": "lightblue",    # Change background color
    "fold": 40,                         # Fold long circuits
    "showindex": True,                  # Show qubit indices
    "compress": False,                  # Don't compress the visualization
    "displaycolor": {                   # Change Hadamard Gate Color and font color
        "h": [
            "#9AA3FF",
            "#406B51"
        ],
        "cx": [
            "#56A179",
            "#DA8B4F"
        ]
    }
}

print("Circuit with custom styling (light blue background):")
circuit.draw(output="mpl", style=style)
```

### 9. 스케일 조정

회로 다이어그램의 스케일을 조정하여 다양한 디스플레이 크기나 문서 제약 조건에 맞출 수 있습니다.

```python
# Scale the circuit to half the normal size
print("Circuit scaled to 50% of normal size:")
circuit.draw(output="mpl", scale=0.5)
```


### 10. circuit_drawer 함수 사용하기

`draw()` 메서드의 대안으로 `circuit_drawer` 함수를 직접 사용할 수 있습니다. 두 접근 방식 모두 동일한 기능을 제공합니다.

```python
from qiskit.visualization import circuit_drawer
 
# Using the functional interface for circuit drawing
print("Using circuit_drawer function directly:")
circuit_drawer(circuit, output="mpl")
```

## 2. 양자 측정 시각화

개요: 이 노트북은 양자 회로 측정을 시각화하는 방법을 다룹니다.

Qiskit을 사용하여 측정을 시각화하는 다양한 방법이 있으며, 결과를 정확하게 표현하기 위해 적절한 시각화를 사용하는 것이 중요합니다.

```python
# Setup: Import necessary libraries
import numpy as np
import matplotlib.pyplot as plt
from qiskit import QuantumCircuit,QuantumRegister, ClassicalRegister, transpile
from qiskit.providers.fake_provider import GenericBackendV2
from qiskit.quantum_info import Statevector

print("Libraries imported successfully.")
```

목표 1: 측정 결과 시각화

Qiskit은 양자 측정 결과와 양자 상태를 분석하기 위한 포괄적인 시각화 도구를 제공합니다.

### 1. 기본 히스토그램 플로팅

`plot_histogram`은 측정 결과의 빈도 분포를 보여주는 측정 결과 시각화에서 가장 일반적인 방법입니다.

```python
from qiskit.visualization import plot_histogram
# Simulated measurement results from quantum circuits
counts1 = {'00': 499, '11': 501}  # First execution: nearly equal distribution
counts2 = {'00': 511, '11': 489}  # Second execution: slight bias toward |00⟩
 
data = [counts1, counts2]
plot_histogram(data)

# This shows the probability distribution of measuring |00⟩ and |11⟩ states
# from two different circuit executions
```

#### 1. 사용자 정의 히스토그램 시각화

히스토그램의 일반적인 사용자 정의 매개변수:
- `title`: 차트 제목
- `figsize`: 인치 단위의 그림 크기(너비, 높이)
- `legend`: 여러 데이터 시리즈의 레이블
- `color`: 각 데이터 시리즈의 사용자 정의 색상
- `filename(str)`: 이미지를 저장할 파일 경로

```python
legend = ['First execution', 'Second execution']
title = 'Bell State Measurement Results'
figsize = (5, 3)
color = ['crimson', 'midnightblue']

plot_histogram(data, 
               legend=legend, 
               title=title, 
               figsize=figsize, 
               color=color)

# Customizations make plots more informative for presentations and publications
```

#### 2. 양자 시각화 유형

- 기본 카운트 시각화
    - `plot_histogram`: 측정 결과 빈도의 막대 차트

    ```python
    # Ideal Bell state measurement results
    counts = {"00": 501, "11": 499}
    plot_histogram(counts)

    # This represents near-perfect Bell state (|00⟩ + |11⟩)/√2 measurements
    # with approximately equal probability for |00⟩ and |11⟩ outcomes
    ```
    
    - `plot_distribution`: Probability distribution visualization
    
    ```python
    from qiskit.visualization import plot_distribution

    counts_data = {
        '00': 245,
        '01': 125,
        '10': 130,
        '11': 500
    }

    plot_distribution(counts_data, 
                        title="Measurement Outcome Distribution",
                        figsize=(8, 5),
                        color='teal',
                        sort='asc')
    ```

    - 여러 분포를 동시에 그래프로 나타낼 수도 있습니다.

    ```python
    # Multiple distributions for comparison
    counts_comparison = [
        {'00': 285, '01': 205, '10': 245, '11': 265},  # First experiment
        {'00': 400, '01': 150, '10': 200, '11': 250},  # Second experiment
        {'00': 350, '01': 200, '10': 220, '11': 230}   # Third experiment
    ]

    plot_distribution(counts_comparison,
                        title="Multiple Experiment Comparison",
                        legend=['Run 1', 'Run 2', 'Run 3'],
                        figsize=(10, 6),
                        color=['#1f77b4', '#ffff0e', "#a02c49"],
                        bar_labels=True)
    ```


- 양자 상태 시각화
    - `plot_bloch_vector`: 블로흐 구에서의 단일 큐비트 상태

    ```python
    from qiskit.visualization import plot_bloch_vector

    # Various single qubit states on Bloch sphere
    bloch_vectors = [1/np.sqrt(2), 1/np.sqrt(2),0]
    title = "XY Superposition"


    plot_bloch_vector(bloch_vectors, 
                        title=title, 
                        coord_type='cartesian')  # Can also use 'spherical'
    # Cartesian: [x, y, z]로 직접 좌표 지정
    # Spherical: [반지름, 극각, 방위각]으로 각도 기반 표현 
    ```

    - `plot_bloch_multivector`: 별도의 블로흐 구에서 여러 큐비트 시각화

    ```python
    from qiskit.visualization import plot_bloch_multivector

    qc = QuantumCircuit(3)
    qc.x(0)
    qc.h(1)
    qc.y(2)
    state = Statevector(qc)

    plot_bloch_multivector(state,figsize=[2,2])

    ```

    - `plot_state_city`: 상태벡터 진폭의 3D 도시 풍경 플롯

    ```python
    from qiskit.visualization import plot_state_city
 
    qc_bell = QuantumCircuit(2)
    qc_bell.h(0)
    qc_bell.cx(0, 1)
    state_bell = Statevector(qc_bell)

    plot_state_city(state_bell, title="Quantum State City Plot")

    # City plots show the magnitude of statevector or density matrix elements
    ```

    - `plot_state_hinton`: 밀도 행렬 시각화를 위한 힌턴 다이어그램

    ```python
    from qiskit.visualization import plot_state_hinton

    plot_state_hinton(state_bell, 
                    title="Bell State - Hinton Diagram",
                    figsize=(8, 6))

    ```

    - `plot_state_paulivec`: 파울리 기댓값 시각화

    ```python
    from qiskit.visualization import plot_state_paulivec

    plot_state_paulivec(state_bell,
                    title="Bell State - Pauli Expectation Values",
                    figsize=(7, 5),
                    color='green')
    ```

    - `plot_state_qsphere`: 양자 상태의 Q-구 표현

    ```python
    from qiskit.visualization import plot_state_qsphere

    plot_state_qsphere(state_bell,figsize=(4,4))
    ```

## 3. 양자 상태 시각화

**개요:** 이 노트북은 양자 상태를 시각화하는 방법을 다룹니다.

양자 상태의 시각화는 Qiskit에서 Bloch 다중 벡터 플롯 및 Q-sphere 표현을 사용하여 수행할 수 있습니다. 이러한 시각화는 다중 큐비트 양자 상태와 그 속성을 직관적으로 이해하는 방법을 제공합니다.

```python
# Setup: Import necessary libraries
from qiskit import QuantumCircuit
from qiskit.quantum_info import Statevector,DensityMatrix
import numpy as np


print("Libraries imported successfully.")
```

### 1. Bloch 다중 벡터 시각화(plot_bloch_multivector)

`plot_bloch_multivector` 함수는 다중-큐비트 시스템의 각 큐비트에 대해 하나씩 여러 개의 Bloch 구를 표시합니다. 각 구는 특정 큐비트에 대한 Pauli 연산자(X, Y, Z)의 기댓값을 보여줍니다.

이는 다중-큐비트 시스템 내 개별 큐비트의 상태를 검사하는 데 가장 효과적입니다. 그러나 큐비트 간의 얽힘이나 위상 관계를 보여줄 수는 없습니다. 주요 사용 사례는 양자 게이트가 회로의 각 큐비트에 어떻게 영향을 미치는지 분석하는 것입니다.

```python
from qiskit.visualization import plot_bloch_multivector

# Create a simple 2-qubit circuit
qc = QuantumCircuit(2)
qc.h(0)    # Apply Hadamard to qubit 0: creates superposition |+⟩ = (|0⟩ + |1⟩)/√2
qc.x(1)
qc.h(1)    

# Get the statevector representing the quantum state
state = Statevector(qc)
print("Statevector:", state)
 
# Visualize each qubit on its own Bloch sphere
plot_bloch_multivector(state, title="2-Qubit State on Bloch Spheres",figsize=(2,2),title_font_size=30)
```

#### 1. 밀도 행렬과 반전된 큐비트 순서를 사용한 Bloch 다중 벡터

밀도 행렬은 혼합 상태(통계적 앙상블)를 나타낼 수 있으며, `reverse_bits`는 큐비트의 표시 순서를 변경합니다.

```python
 # Create a more complex 3-qubit circuit
qc = QuantumCircuit(3)
qc.h([0, 1,2])    # Hadamard on all qubits: creates |+⟩ ⊗ |+⟩ ⊗ |+⟩ 
qc.t(1)         # T gate on qubit 1: adds π/4 phase (rotation around Z-axis)
qc.s(0)         # S gate on qubit 0: adds π/2 phase (rotation around Z-axis)
qc.z(2)         # Z gate on qubit 2
qc.cx(0, 1)     # CNOT gate: creates entanglement between qubits
qc.cx(1, 2)     # CNOT gate: creates entanglement between qubits
 
# Use DensityMatrix instead of Statevector
# Density matrices can represent both pure and mixed states
matrix = DensityMatrix(qc)
print("Density matrix shape:", matrix.dim)
 
# Visualize with reversed bit order and custom title and font size
plot_bloch_multivector(matrix, 
                     title='Bloch Spheres from Dnesity Matrix', 
                     reverse_bits=True,
                     font_size=20)
```

### 2. Q-Sphere 시각화(plot_state_qsphere)

Q-sphere는 진폭과 위상을 나타내는 색상과 화살표를 통해 계산 기저 상태를 구의 점에 매핑하여 다중-큐비트 상태를 간결하게 시각화하는 방법을 제공합니다.

이는 완전한 다중-큐비트 양자 상태를 검사하는 데 가장 효과적이며, 큐비트 간의 얽힘과 위상 관계를 보여줄 수 있습니다. 그러나 큐비트 수가 증가함에 따라 시각화가 복잡해집니다.

```python
from qiskit import QuantumCircuit
from qiskit.quantum_info import Statevector
from qiskit.visualization import plot_state_qsphere
 
# Create a circuit that produces an interesting quantum state
qc = QuantumCircuit(2)
qc.h(0)        # Hadamard: creates superposition
qc.z(0)        # Z gate: adds phase flip to |1⟩ component
qc.x(1)        # X gate: |0⟩ → |1⟩
qc.cx(0, 1)    # CNOT: creates controlled operation
 
# Get the final state
state = Statevector(qc)
print("Final statevector:")
print(state)
 
# Visualize on Q-sphere with enhanced options
plot_state_qsphere(state, 
                  use_degrees=False,           # Show phases in radians
                  show_state_phases=True,     # Display phase arrows
                  show_state_labels =True)

# - Points on sphere represent computational basis states |00⟩, |01⟩, |10⟩, |11⟩
# - Points size shows magnitude of each amplitude
# - Colors indicate different phase regions
# 크기 = 확률
# 색/화살표 = 위상 (핵심)
```

#### 1. 밀도 행렬 및 회전 게이트를 사용한 고급 Q-Sphere

이 예제는 사용자 정의 회전을 사용하여 더 복잡한 상태를 보여주며, 밀도 행렬을 사용한 Q-sphere 시각화를 보여줍니다.

```python
from qiskit.visualization import plot_state_qsphere

# - Multiple basis states have non-zero amplitudes
# - Phase colors show complex phases introduced by rotation gates
# - Degrees make phase values more intuitive (0° = +1, 180° = -1, etc.)

# Create a circuit with various quantum gates
qc = QuantumCircuit(2)
qc.h([0, 1])        # Hadamard on both: creates equal superposition
qc.cz(0, 1)         # Controlled-Z: entangles qubits with phase kick
qc.ry(np.pi/3, 0)   # Rotation around Y-axis by 60° on qubit 0
qc.rx(np.pi/6, 1)   # Rotation around X-axis by 30° on qubit 1
qc.z(1)             # Z gate on qubit 1: global phase shift
 
# Use density matrix representation
matrix = DensityMatrix(qc)
print("Circuit creates a complex entangled state with custom rotations")
 
# Enhanced Q-sphere visualization
plot_state_qsphere(matrix,
     show_state_phases=True,    # Show phase arrows for each basis state
     use_degrees=True,          # Display phases in degrees (0-360°)
     show_state_labels =True)
```
