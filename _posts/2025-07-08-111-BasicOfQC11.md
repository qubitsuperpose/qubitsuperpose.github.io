# Qiskit 구현

이 강의에서는 **얽힘의 실제 활용**에 대한 강의에서 다룬 아이디어 중 일부를 Qiskit을 사용하여 구현합니다.

```python
from qiskit import __version__
 
print(__version__)
```

출력: 2.1.2

```python
from qiskit import QuantumCircuit, QuantumRegister, ClassicalRegister
from qiskit_aer import AerSimulator
from qiskit.visualization import plot_histogram, array_to_latex
from qiskit.result import marginal_distribution
from qiskit.circuit.library import UGate
from numpy import pi, random
```

텔레포테이션 프로토콜의 양자 회로 구현은 다음과 같습니다.

```python
qubit = QuantumRegister(1, "Q")
ebit0 = QuantumRegister(1, "A")
ebit1 = QuantumRegister(1, "B")
a = ClassicalRegister(1, "a")
b = ClassicalRegister(1, "b")
 
protocol = QuantumCircuit(qubit, ebit0, ebit1, a, b)
 
# Prepare ebit used for teleportation
protocol.h(ebit0)
protocol.cx(ebit0, ebit1)
protocol.barrier()
 
# Alice's operations
protocol.cx(qubit, ebit0)
protocol.h(qubit)
protocol.barrier()
 
# Alice measures and sends classical bits to Bob
protocol.measure(ebit0, a)
protocol.measure(qubit, b)
protocol.barrier()
 
# Bob uses the classical bits to conditionally apply gates
with protocol.if_test((a, 1)):
    protocol.x(ebit1)
with protocol.if_test((b, 1)):
    protocol.z(ebit1)
 
display(protocol.draw(output="mpl"))
```

**회로는 우리가 이전 강의에서 아직 보지 못했던 Qiskit의 몇 가지 기능, 즉 $\text{barrier}$ 및 $\text{if\_test}$ 함수를 사용합니다**. $\text{barrier}$ 함수는 회로 다이어그램을 더 읽기 쉽게 만드는 시각적 분리를 생성하며, 또한 회로를 실제 하드웨어에서 실행할 때 컴파일하는 동안 Qiskit이 배리어를 넘어 다양한 단순화 및 최적화를 수행하는 것을 방지합니다. $\text{if\_test}$ 함수는 **고전적인 비트나 레지스터에 따라 조건부로 연산을 적용**합니다.

회로는 먼저 $(\mathsf{A}, \mathsf{B})$를 $\vert \phi^+\rangle$ 상태로 초기화하며 (이는 프로토콜 자체의 일부는 아닙니다), 이어서 **앨리스의 연산, 그녀의 측정, 그리고 마지막으로 밥의 연산**이 뒤따릅니다. 프로토콜이 올바르게 작동하는지 테스트하기 위해, 우리는 무작위로 생성된 단일 큐비트 게이트를 $\mathsf{Q}$의 초기화된 $\vert 0\rangle$ 상태에 적용하여 텔레포트될 무작위 양자 상태 벡터를 얻을 것입니다. 프로토콜이 실행된 후 그 게이트의 역(즉, 켤레 전치)을 $\mathsf{B}$에 적용함으로써, 우리는 측정을 통해 상태가 $\vert 0\rangle$ 상태로 돌아왔음을 확인하여 텔레포트되었는지 검증할 수 있습니다.

먼저 우리는 유니터리 큐비트 게이트를 무작위로 선택할 것입니다.

```python
random_gate = UGate(
    theta=random.random() * 2 * pi,
    phi=random.random() * 2 * pi,
    lam=random.random() * 2 * pi,
)
 
display(array_to_latex(random_gate.to_matrix()))
```

 0.0603319186 + 0.1296609988 i & -0.8319925233 + 0.5360378028 i  \\
 \end{bmatrix}
$$

이제 우리는 무작위 게이트를 $\mathsf{Q}$에 먼저 적용하고, 이어서 텔레포테이션 회로를 실행한 다음, 마지막으로 무작위 게이트의 역을 큐비트 $\mathsf{B}$에 적용하고 측정하는 새로운 테스트 회로를 만들 것입니다. **결과는 확실하게 $0$이 되어야 합니다**.

```python
# Create a new circuit including the same bits and qubits used in the
# teleportation protocol.
 
test = QuantumCircuit(qubit, ebit0, ebit1, a, b)
 
# Start with the randomly selected gate on Q
 
test.append(random_gate, qubit)
test.barrier()
 
# Append the entire teleportation protocol from above.
 
test = test.compose(protocol)
test.barrier()
 
# Finally, apply the inverse of the random unitary to B and measure.
 
test.append(random_gate.inverse(), ebit1)
result = ClassicalRegister(1, "Result")
test.add_register(result)
test.measure(ebit1, result)
 
display(test.draw(output="mpl"))
```

## 1. 초밀집 코딩
초밀집 코딩(Superdense coding)은 어떤 면에서 **텔레포테이션과는 상보적인 목표를 달성**하는 프로토콜입니다. 초밀집 코딩은 (하나의 얽힘 비트(e-bit)를 소모하여) 두 비트의 고전적 통신을 사용하여 하나의 큐비트를 전송하는 대신, **(역시 하나의 얽힘 비트를 소모하여) 하나의 큐비트 양자 통신을 사용하여 두 비트의 고전적 통신을 전송**할 수 있도록 합니다.

더 자세히 설명하자면, 우리에게는 하나의 얽힘 비트를 공유하는 송신자(앨리스)와 수신자(밥)가 있습니다. 이 강의에서 통용되는 관례에 따르면, 이는 앨리스가 큐비트 $\mathsf{A}$를, 밥이 큐비트 $\mathsf{B}$를 보유하고 있으며, $(\mathsf{A}, \mathsf{B})$ 쌍은 $\vert\phi^+\rangle$ 상태에 있다는 의미입니다. 앨리스는 밥에게 두 개의 고전적 비트($c$와 $d$로 표시)를 전송하기를 원하며, 그녀는 하나의 큐비트를 그에게 보냄으로써 이를 달성할 것입니다.

이러한 업적은 텔레포테이션이 달성하는 것보다 덜 흥미롭게 보는 것이 합리적입니다. 가까운 미래까지 큐비트를 보내는 것은 고전적 비트를 보내는 것보다 훨씬 더 어려울 가능성이 높기 때문에, 얽힘 비트까지 소모하면서 하나의 양자 통신 큐비트를 두 비트의 고전적 통신과 교환하는 것은 거의 가치가 없어 보입니다. 하지만 이것이 초밀집 코딩이 흥미롭지 않다는 것을 의미하지는 않습니다. 분명히 흥미로운 점이 있습니다. 

이 강의의 주제에 맞게, 초밀집 코딩이 흥미로운 한 가지 이유는 **얽힘의 구체적이고 (정보 이론의 맥락에서) 꽤 놀라운 사용을 입증**하기 때문입니다. 양자 정보 이론의 유명한 정리인 **'홀레보 정리(Holevo's theorem)'**는 공유된 얽힘 상태를 사용하지 않고는 단일 큐비트를 전송하여 한 비트 이상의 고전적 정보를 통신하는 것이 불가능함을 시사합니다. (홀레보 정리는 이보다 더 일반적입니다. 그것의 정확한 진술은 기술적이며 설명이 필요하지만, 이것이 그 결과 중 하나입니다). 따라서 초밀집 코딩을 통해 공유된 얽힘은 큐비트 전송의 고전적 정보 전달 용량을 효과적으로 *두 배로* 늘릴 수 있게 합니다.

### 1.1 프로토콜
다음 양자 회로 다이어그램은 초밀집 코딩 프로토콜을 설명합니다.

<img src="https://quantum.cloud.ibm.com/assets-docs-learning/_next/image?url=%2Flearning%2Fimages%2Fcourses%2Fbasics-of-quantum-information%2Fentanglement-in-action%2Fqiskit-implementation%2Fsuperdense-coding.avif&w=3840&q=75" style="width:70%">

말로 설명하면 앨리스가 하는 일은 다음과 같습니다.

1\.**만약 $d=1$이면, 앨리스는 그녀의 큐비트 $\mathsf{A}$에 $\mathsf{Z}$ 게이트를 수행합니다 (그리고 $d=0$이면 수행하지 않습니다)**.

2\.**만약 $c=1$이면, 앨리스는 그녀의 큐비트 $\mathsf{A}$에 $\mathsf{X}$ 게이트를 수행합니다 (그리고 $c=0$이면 수행하지 않습니다)**.

그런 다음 앨리스는 그녀의 큐비트 $\mathsf{A}$를 밥에게 보냅니다.

밥이 큐비트 $\mathsf{A}$를 받았을 때 하는 일은, 먼저 $\mathsf{A}$를 컨트롤로, $\mathsf{B}$를 타깃으로 하는 **제어-NOT 게이트를 수행**한 다음, $\mathsf{A}$에 **하마르드 게이트를 적용**하는 것입니다. 그런 다음 그는 $\mathsf{B}$를 측정하여 $c$를 얻고, $\mathsf{A}$를 측정하여 $d$를 얻으며, 두 경우 모두 표준 기저 측정을 사용합니다.

### 1.2 분석

이 프로토콜의 이면의 아이디어는 간단합니다. 앨리스는 자신이 밥과 공유하고 싶은 **벨 상태**를 효과적으로 선택하고, 그녀의 큐비트를 밥에게 보냅니다. 그러면 밥은 앨리스가 어떤 벨 상태를 선택했는지 측정하여 결정합니다. 

즉, 그들은 처음에 $\vert\phi^+\rangle$ 상태를 공유하며, 비트 $c$와 $d$에 따라 앨리스는 이 상태를 그대로 두거나, 또는 그녀의 큐비트 $\mathsf{A}$에 $\mathsf{I}$, $\mathsf{X}$, $\mathsf{Z}$, 또는 $\mathsf{XZ}$를 적용하여 다른 벨 상태 중 하나로 변경합니다.

<div>
$$
\begin{aligned}
(\mathbb{I} \otimes \mathbb{I}) \vert \phi^+ \rangle & = \vert \phi^+\rangle \\
(\mathbb{I} \otimes Z) \vert \phi^+ \rangle & = \vert \phi^-\rangle \\
(\mathbb{I} \otimes X) \vert \phi^+ \rangle & = \vert \psi^+\rangle \\
(\mathbb{I} \otimes XZ) \vert \phi^+ \rangle & = \vert \psi^-\rangle
\end{aligned}
$$
</div>

밥의 연산은 네 가지 벨 상태에 다음과 같은 영향을 미칩니다. 

<div>
$$
\begin{aligned}
\vert \phi^+\rangle & \mapsto \vert 00\rangle\\
\vert \phi^-\rangle & \mapsto \vert 01\rangle\\
\vert \psi^+\rangle & \mapsto \vert 10\rangle\\
\vert \psi^-\rangle & \mapsto -\vert 11\rangle\\
\end{aligned}
$$
</div>

이것은 밥의 연산을 이러한 상태에 대해 하나씩 계산함으로써 직접 확인할 수 있습니다.

따라서 밥이 측정을 수행할 때, 그는 앨리스가 어떤 벨 상태를 선택했는지 결정할 수 있습니다. 
프로토콜이 올바르게 작동하는지 확인하는 것은 각 사례를 확인하는 문제입니다.

- 만약 $cd = 00$이라면, 밥이 $\mathsf{A}$를 받을 때 $(\mathsf{B}, \mathsf{A})$의 상태는 $\vert \phi^+\rangle$입니다. 그는 이 상태를 $\vert 00\rangle$로 변환하고 $cd = 00$을 얻습니다.
- 만약 $cd = 01$이라면, 밥이 $\mathsf{A}$를 받을 때 $(\mathsf{B}, \mathsf{A})$의 상태는 $\vert \phi^-\rangle$입니다. 그는 이 상태를 $\vert 01\rangle$로 변환하고 $cd = 01$을 얻습니다.
- 만약 $cd = 10$이라면, 밥이 $\mathsf{A}$를 받을 때 $(\mathsf{B}, \mathsf{A})$의 상태는 $\vert \psi^+\rangle$입니다. 그는 이 상태를 $\vert 10\rangle$으로 변환하고 $cd = 10$을 얻습니다.
- 만약 $cd = 11$이라면, 밥이 $\mathsf{A}$를 받을 때 $(\mathsf{B}, \mathsf{A})$의 상태는 $\vert \psi^-\rangle$입니다. 그는 이 상태를 $-\vert 11\rangle$로 변환하고 $cd = 11$을 얻습니다. (음수 일($-1$) 위상 인자는 여기서는 효과가 없습니다).

### 1.3 초밀집 코딩 구현
여기에 전송할 비트에 따라 회로 자체를 지정하는 초밀집 코딩의 간단한 구현이 있습니다. 먼저 전송할 두 비트를 선택할 것입니다. (나중에는 무작위로 선택하겠지만, 지금은 임의로 선택할 것입니다).

```python
c = "1"
d = "0"
```

이제 그에 따라 회로를 구축할 것입니다. 여기서는 Qiskit이 큐비트에 대해 기본 이름을 사용하도록 허용할 것입니다. 즉, 상단 큐비트에는 $\mathsf{q}_0$을, 하단 큐비트에는 $\mathsf{q}_1$을 사용합니다.

```python
protocol = QuantumCircuit(2)
 
# Prepare ebit used for superdense coding
protocol.h(0)
protocol.cx(0, 1)
protocol.barrier()
 
# Alice's operations
if d == "1":
    protocol.z(0)
if c == "1":
    protocol.x(0)
protocol.barrier()
 
# Bob's actions
protocol.cx(0, 1)
protocol.h(0)
protocol.measure_all()
 
display(protocol.draw(output="mpl"))
```

여기서 $\text{measure\_all}$ 함수를 제외하고는 새로운 것이 많지 않습니다. 이 함수는 **모든 큐비트를 측정**하고 그 결과를 단일 고전적 레지스터에 넣습니다 (이 경우 두 비트를 가집니다).

Aer 시뮬레이터를 실행하면 예상된 출력이 나옵니다.

```python
result = AerSimulator().run(protocol).result()
statistics = result.get_counts()
 
for outcome, frequency in statistics.items():
    print(f"Measured {outcome} with frequency {frequency}")
 
display(plot_histogram(statistics))
```

이제 우리는 무작위 비트 생성기로서 추가 큐비트, 즉 본질적으로 **공정한 동전을 던지는 역할을 하는 큐비트**를 사용할 것입니다. 우리는 이 큐비트를 사용하여 $c$와 $d$를 무작위로 선택한 다음 초밀집 코딩 프로토콜을 실행할 것입니다.

```python
rbg = QuantumRegister(1, "coin")
ebit0 = QuantumRegister(1, "A")
ebit1 = QuantumRegister(1, "B")
 
Alice_c = ClassicalRegister(1, "Alice c")
Alice_d = ClassicalRegister(1, "Alice d")
 
test = QuantumCircuit(rbg, ebit0, ebit1, Alice_d, Alice_c)
 
# Initialize the ebit
test.h(ebit0)
test.cx(ebit0, ebit1)
test.barrier()
 
# Use the 'coin' qubit twice to generate Alice's bits c and d.
test.h(rbg)
test.measure(rbg, Alice_c)
test.h(rbg)
test.measure(rbg, Alice_d)
test.barrier()
 
# Now the protocol runs, starting with Alice's actions, which depend
# on her bits.
with test.if_test((Alice_d, 1), label="Z"):
    test.z(ebit0)
with test.if_test((Alice_c, 1), label="X"):
    test.x(ebit0)
test.barrier()
 
# Bob's actions
test.cx(ebit0, ebit1)
test.h(ebit0)
test.barrier()
 
Bob_c = ClassicalRegister(1, "Bob c")
Bob_d = ClassicalRegister(1, "Bob d")
test.add_register(Bob_d)
test.add_register(Bob_c)
test.measure(ebit0, Bob_d)
test.measure(ebit1, Bob_c)
```

## 2. CHSH 게임

이 강의에서 논의할 마지막 예시는 프로토콜이 아니라, **$\text{CHSH}$ 게임**으로 알려진 *게임*입니다.

이러한 맥락에서 우리가 게임을 언급할 때는 재미나 오락을 위해 플레이하도록 고안된 것을 이야기하는 것이 아니라, **'게임 이론(game theory)'**의 의미에서 수학적 추상화를 이야기하는 것입니다. 게임의 수학적 추상화는 예를 들어 경제학과 컴퓨터 과학에서 연구되며, 매혹적이고 유용합니다.

$\text{CHSH}$라는 글자는 1969년에 이 예시를 처음 기술한 저자들, **존 클라우저(John Clauser), 마이클 혼(Michael Horne), 애브너 시모니(Abner Shimony), 리처드 홀트(Richard Holt)**를 나타냅니다. 그들은 이 예시를 게임이 아닌 실험으로 설명했습니다. 그러나 이를 게임으로 설명하는 것은 자연스럽고 직관적입니다. 

$\text{CHSH}$ 게임은 **'비국소적 게임(nonlocal games)'**으로 알려진 게임 부류에 속합니다. 비국소적 게임은 엄청나게 흥미롭고, 물리학, 컴퓨터 과학 및 수학과 깊은 관련이 있으며, 아직 해결되지 않은 미스터리를 담고 있습니다. 우리는 이 섹션을 비국소적 게임이 무엇인지 설명하는 것으로 시작한 다음, $\text{CHSH}$ 게임과 그것을 흥미롭게 만드는 요소에 집중할 것입니다.

### 2.1 비국소적 게임

비국소적 게임은 두 플레이어, **앨리스와 밥이 특정 결과를 달성하기 위해 협력하는 *협력 게임***입니다. 이 게임은 앨리스와 밥에게 알려진 엄격한 지침에 따라 행동하는 *심판(referee)*에 의해 운영됩니다.

앨리스와 밥은 원하는 방식으로 게임을 준비할 수 있지만, 일단 게임이 시작되면 **그들은 통신이 금지**됩니다. 우리는 이 게임이 어떤 종류의 보안 시설에서 진행된다고 상상할 수 있습니다. 마치 심판이 형사의 역할을 하고 앨리스와 밥이 다른 방에서 심문을 받는 용의자인 것처럼 말입니다. 그러나 이 설정을 생각하는 또 다른 방법은 앨리스와 밥이 광활한 거리에 의해 분리되어 있고, 게임 실행 시간 내에는 빛의 속도로 인해 통신이 금지된다는 것입니다. 즉, 앨리스가 밥에게 메시지를 보내려고 시도하면, 그가 메시지를 받을 때쯤에는 게임이 끝날 것이며, 그 반대도 마찬가지입니다.

비국소적 게임이 작동하는 방식은 심판이 먼저 앨리스와 밥에게 각각 질문 하나씩을 던지는 것입니다. 우리는 문자 $x$를 앨리스의 질문을 나타내는 데 사용할 것이고, $y$를 밥의 질문을 나타내는 데 사용할 것입니다. 여기서 우리는 $x$와 $y$를 고전적 상태로 생각하고 있으며, $\text{CHSH}$ 게임에서는 $x$와 $y$가 비트입니다.

심판은 이러한 질문을 선택하기 위해 **무작위성**을 사용합니다. 정확히 말하면, 가능한 각 질문 쌍 $(x, y)$와 관련된 어떤 확률 $p(x, y)$가 있으며, 심판은 게임이 진행될 때 이 방식으로 질문을 무작위로 선택하기로 약속합니다. 앨리스와 밥을 포함한 모든 사람이 이 확률을 알고 있지만, 게임이 시작될 때까지는 어떤 쌍 $(x, y)$가 구체적으로 선택될지 아무도 모릅니다.

앨리스와 밥이 질문을 받은 후, 그들은 답변을 제공해야 합니다. 앨리스의 답변은 $a$이고 밥의 답변은 $b$입니다. 다시 말하지만, 이는 일반적으로 고전적 상태이며 $\text{CHSH}$ 게임에서는 비트입니다.

이 시점에서 심판은 결정을 내립니다. 앨리스와 밥은 정해진 규칙 세트에 따라 답변 쌍 $(a, b)$가 질문 쌍 $(x, y)$에 대해 올바른 것으로 간주되는지 여부에 따라 *승리*하거나 *패배*합니다. 규칙이 다르면 다른 게임을 의미하며, $\text{CHSH}$ 게임에 대한 규칙은 특히 이 섹션 다음에 설명되어 있습니다. 이미 언급되었듯이, 규칙은 모두에게 알려져 있습니다.

다음 다이어그램은 상호 작용을 그래픽으로 표현한 것입니다. 

<img src="https://quantum.cloud.ibm.com/assets-docs-learning/_next/image?url=%2Flearning%2Fimages%2Fcourses%2Fbasics-of-quantum-information%2Fentanglement-in-action%2Fqiskit-implementation%2Fnonlocal-game.avif&w=3840&q=75" style="width:60%">

어떤 질문이 주어질지에 대한 불확실성, 그리고 특히 각 플레이어가 상대방의 질문을 모른다는 사실이 앨리스와 밥에게 비국소적 게임을 도전적으로 만듭니다. 마치 다른 방에서 공모하는 용의자들이 이야기를 일관성 있게 유지하려고 노력하는 것과 같습니다. 

심판에 대한 정확한 설명은 비국소적 게임의 인스턴스를 정의합니다. 여기에는 각 질문 쌍에 대한 확률 $p(x, y)$ 사양과 가능한 각 질문 쌍 $(x, y)$에 대해 각 답변 쌍 $(a, b)$가 승리 또는 패배하는지 결정하는 규칙이 포함됩니다.

우리는 잠시 후 $\text{CHSH}$ 게임을 살펴볼 것이지만, 그 전에 다른 비국소적 게임을 고려하는 것도 흥미롭다는 것을 간략하게 인정합시다. 사실, 그것은 매우 흥미롭습니다. 앨리스와 밥이 얽힘을 사용하여 얼마나 잘 플레이할 수 있는지 현재 알려지지 않은 아주 간단한 비국소적 게임도 있습니다. 설정은 간단하지만 복잡성이 작용하고 있으며, 일부 게임의 경우 앨리스와 밥을 위한 최상의 또는 거의 최상의 전략을 계산하는 것이 불가능할 정도로 어려울 수 있습니다. 이것이 비국소적 게임 모델의 놀랍도록 비직관적인 특성입니다.

### 2.2 $\text{CHSH}$ 게임 설명

여기 $\text{CHSH}$ 게임에 대한 정확한 설명이 있습니다. 여기서 (위와 같이) $x$는 앨리스의 질문, $y$는 밥의 질문, $a$는 앨리스의 답변, $b$는 밥의 답변입니다.

- 질문과 답변은 모두 비트입니다. 
- 심판은 질문 $(x, y)$를 *균일하게 무작위로* 선택합니다. 즉, 네 가지 가능한 경우, $(0, 0)$, $(0, 1)$, $(1, 0)$, 그리고 $(1, 1)$ 각각이 $1/4$의 확률로 선택됩니다.
- 답변 $(a, b)$는 질문 $(x, y)$에 대해 $a \oplus b = x \wedge y$이면 *승리*하고, 그렇지 않으면 *패배*합니다. 다음 표는 각 질문 쌍 $(x, y)$에 대한 답변 $(a, b)$의 승리 및 패배 조건을 나열하여 이 규칙을 표현합니다.

<p>$$
\begin{array}{ccc}
(x,y) & \text{win} & \text{lose} \\[1mm]\hline
\rule{0mm}{4mm}(0,0) & a = b & a \neq b \\[1mm]
(0,1) & a = b & a \neq b \\[1mm]
(1,0) & a = b & a \neq b \\[1mm]
(1,1) & a \neq b & a = b
\end{array}
$$</p>

### 2.3 고전적 전략의 한계

이제 $\text{CHSH}$ 게임에서 앨리스와 밥의 전략을 고려해 봅시다. *고전적* 전략부터 시작하겠습니다.

1\. 결정론적 전략

우리는 *결정론적* 전략부터 시작할 것입니다. 여기서 앨리스의 답변 $a$는 그녀가 받는 질문 $x$의 함수이며, 마찬가지로 밥의 답변 $b$는 그가 받는 질문 $y$의 함수입니다. 따라서, 예를 들어, 앨리스의 질문이 $0$일 때 앨리스의 답변을 $a(0)$으로, 질문이 $1$일 때 앨리스의 답변을 $a(1)$로 나타낼 수 있습니다.

어떤 결정론적 전략도 $\text{CHSH}$ 게임에서 매번 이길 수는 없습니다. 이를 이해하는 한 가지 방법은 가능한 모든 결정론적 전략을 하나씩 검토하고 네 가지 가능한 질문 쌍 중 적어도 하나에 대해서는 모든 전략이 패배하는지 확인하는 것입니다. 앨리스와 밥은 각각 1비트에서 1비트로 가는 네 가지 가능한 함수를 선택할 수 있으며 (이는 단일 시스템에 대한 강의에서 다루었습니다), 따라서 총 16가지의 결정론적 전략을 확인해야 합니다. 

우리는 또한 이것을 분석적으로 추론할 수 있습니다. 만약 앨리스와 밥의 전략이 $(x, y) = (0, 0)$일 때 이긴다면, $a(0) = b(0)$이어야 합니다. 만약 그들의 전략이 $(x, y) = (0, 1)$일 때 이긴다면, $a(0) = b(1)$이고, 마찬가지로, 전략이 $(x, y) = (1, 0)$에 대해 이긴다면 $a(1) = b(0)$입니다. 따라서, 그들의 전략이 세가지 가능성 모두에서 이길 경우는,

$b(1) = a(0) = b(0) = a(1).$

이는 그 전략이 마지막 경우인 $(x, y) = (1, 1)$에서는 패배한다는 것을 의미하며, 이 경우 승리하려면 $a(1) \neq b(1)$이 요구됩니다. 따라서 매번 승리하는 결정론적 전략은 있을 수 없습니다. 

반면에, 네 가지 경우 중 세 가지에서 승리하는 결정론적 전략을 찾기는 쉽습니다. 예를 들어, $a(0)=a(1)=b(0)=b(1)=0$. 이로부터 우리는 앨리스와 밥이 결정론적 전략을 사용하여 승리할 수 있는 최대 확률은 $3/4$이라는 결론을 내립니다.

2\. 확률론적 전략

우리가 방금 결론 내렸듯이, 앨리스와 밥은 결정론적 전략을 사용하여 $\text{CHSH}$ 게임에서 $75\%$ 이상 승리할 수 없습니다. 하지만 확률론적 전략은 어떨까요? 무작위성, 즉 그들의 무작위 선택이 상관관계가 있는 *공유된 무작위성*을 사용하는 것이 앨리스와 밥에게 도움이 될 수 있을까요?

확률론적 전략은 앨리스와 밥이 승리할 확률을 높이는 데 전혀 도움이 되지 않는 것으로 밝혀졌습니다. 이는 모든 확률론적 전략이 결정론적 전략의 무작위 선택으로 대체하여 볼 수 있기 때문입니다. 이는 (단일 시스템 강의에서 언급되었듯이) 확률론적 연산이 결정론적 연산의 무작위 선택으로 볼 수 있는 것과 같습니다. 평균은 최댓값보다 클 수 없으므로, 확률론적 전략은 전체 승리 확률 측면에서 어떤 이점도 제공하지 않는다는 결론이 나옵니다.

따라서 **$3/4$의 확률로 승리하는 것이 앨리스와 밥이 결정론적이든 확률론적이든 고전적 전략을 사용하여 할 수 있는 최선**입니다.

### 2.4 $\text{CHSH}$ 게임 전략

이 시점에서 던질 수 있는 자연스러운 질문은 앨리스와 밥이 *양자* 전략을 사용하여 더 잘할 수 있는지 여부입니다. 특히, 다음 그림이 제시하는 것처럼 게임을 하기 전에 준비했을 수 있는 얽힌 양자 상태를 공유한다면, 그들은 승리 확률을 높일 수 있을까요?

<img src="https://quantum.cloud.ibm.com/assets-docs-learning/_next/image?url=%2Flearning%2Fimages%2Fcourses%2Fbasics-of-quantum-information%2Fentanglement-in-action%2Fqiskit-implementation%2Fnonlocal-game-entanglement.avif&w=3840&q=75" style="width:60%">

답변은 '예'이며, 이것이 이 예시의 주요 요점이며 이 예시가 매우 흥미로운 이유입니다. 그렇다면 앨리스와 밥이 얽힘을 사용하여 이 게임에서 정확히 어떻게 더 잘할 수 있는지 봅시다.

1\. 필요한 벡터 및 행렬

우리가 해야 할 첫 번째 일은 각 실수 $\theta$ (라디안으로 측정된 각도로 생각할 것입니다)에 대해 큐비트 상태 벡터 $\vert \psi_{\theta}\rangle$를 다음과 같이 정의하는 것입니다.

$\vert\psi_{\theta}\rangle = \cos(\theta)\vert 0\rangle + \sin(\theta) \vert 1\rangle$

몇가지 간단한 예: 
<p>
$$\begin{aligned}
  \vert\psi_{0}\rangle & = \vert 0\rangle \\
  \vert\psi_{\pi/2}\rangle & = \vert 1\rangle \\
  \vert\psi_{\pi/4}\rangle & = \vert + \rangle \\
  \vert\psi_{-\pi/4}\rangle & = \vert - \rangle
\end{aligned}$$
</p>

또한 아래 분석에서 발생하는 다음과 같은 예도 있다.
<p>
$$
\begin{aligned}
  \vert\psi_{-\pi/8}\rangle & = \frac{\sqrt{2 + \sqrt{2}}}{2}\vert 0\rangle -\frac{\sqrt{2 - \sqrt{2}}}{2}\vert 1\rangle \\[1mm]
  \vert\psi_{\pi/8}\rangle & = \frac{\sqrt{2 + \sqrt{2}}}{2}\vert 0\rangle + \frac{\sqrt{2 - \sqrt{2}}}{2}\vert 1\rangle \\[1mm]
  \vert\psi_{3\pi/8}\rangle & = \frac{\sqrt{2 - \sqrt{2}}}{2}\vert 0\rangle + \frac{\sqrt{2 + \sqrt{2}}}{2}\vert 1\rangle \\[1mm]
  \vert\psi_{5\pi/8}\rangle & = -\frac{\sqrt{2 - \sqrt{2}}}{2}\vert 0\rangle + \frac{\sqrt{2 + \sqrt{2}}}{2}\vert 1\rangle
\end{aligned}
$$
</p>

일반적인 형태를 보면, 이러한 벡터 중 임의의 두 벡터 사이의 내적은 다음과 같은 공식을 가집니다. 
<p>
$$
\langle \psi_{\alpha} \vert \psi_{\beta} \rangle
= \cos(\alpha)\cos(\beta) + \sin(\alpha)\sin(\beta)
= \cos(\alpha-\beta).
\tag{3}
$$
</p>

자세히 말하면, 이러한 벡터에는 실수 항목만 있으므로 복소수 켤레에 대해 걱정할 필요가 없습니다. 내적은 코사인의 곱과 사인의 곱의 합입니다. 삼각법의 *각 합 공식* 중 하나를 사용하면 위와 같은 단순화가 가능합니다. 이 공식은 실수 단위 벡터 사이의 내적을 그들 사이의 각도의 코사인으로 기하학적으로 해석합니다.

이러한 벡터 중 임의의 두 벡터의 *텐서 곱*과 $\vert \phi^+\rangle$ 상태의 내적을 계산하면, 분모에 $\sqrt{2}$가 있다는 점을 제외하고는 유사한 표현을 얻습니다.

<p>
$$
\langle \psi_{\alpha} \otimes \psi_{\beta} \vert \phi^+ \rangle
= \frac{\cos(\alpha)\cos(\beta) + \sin(\alpha)\sin(\beta)}{\sqrt{2}}
= \frac{\cos(\alpha-\beta)}{\sqrt{2}}.
\tag{4}
$$
</p>

이 특정 내적에 대한 우리의 관심은 곧 명확해질 것이지만, 지금은 단순히 이것을 공식으로 관찰하고 있습니다.

다음으로, 각도 $\theta$에 대해 유니터리 행렬 $U_{\theta}$를 다음과 같이 정의합니다. 
<p>
$$
U_{\theta} = \vert 0 \rangle \langle \psi_{\theta} \vert + \vert 1\rangle\langle \psi_{\theta+\pi/2} \vert
$$
</p>

직관적으로 말하면, 이 행렬은 $\vert\psi_{\theta}\rangle$ 를 $\vert 0\rangle$ 로, 그리고 $\vert \psi_{\theta + \pi/2}\rangle$를 $\vert 1\rangle$ 로 변환합니다. 이것이 유니터리 행렬인지 확인하기 위한 핵심 관찰은, 벡터 $\vert\psi_{\theta}\rangle$ 와 $\vert\psi_{\theta + \pi/2}\rangle$ 가 모든 각도 $\theta$ 에 대해 직교한다는 것입니다.

<p>$$
\langle \psi_{\theta} \vert \psi_{\theta + \pi/2} \rangle = \cos(\pi/2) = 0.
$$</p>

따라서, 우리는 다음을 발견한다.

<p>$$
\begin{aligned}
U_{\theta} U_{\theta}^{\dagger}
& = \bigl(\vert 0 \rangle \langle \psi_{\theta} \vert + \vert 1\rangle\langle \psi_{\theta+\pi/2} \vert\bigr)
\bigl(\vert \psi_{\theta} \rangle \langle 0 \vert + \vert \psi_{\theta+\pi/2}\rangle\langle 1 \vert\bigr) \\[1mm]
& =
\vert 0 \rangle \langle \psi_{\theta} \vert \psi_{\theta} \rangle \langle 0 \vert
+ \vert 0 \rangle \langle \psi_{\theta} \vert \psi_{\theta+\pi/2} \rangle \langle 1 \vert
+ \vert 1 \rangle \langle \psi_{\theta+\pi/2} \vert \psi_{\theta} \rangle \langle 0 \vert
+ \vert 1 \rangle \langle \psi_{\theta+\pi/2} \vert \psi_{\theta+\pi/2} \rangle \langle 1 \vert \\[1mm]
& =
\vert 0 \rangle \langle 0 \vert + \vert 1 \rangle \langle 1 \vert\\[1mm]
& = \mathbb{I}.
\end{aligned}
$$</p>

우리는 대안적으로 이 행렬을 다음과 같이 명시적으로 작성할 수 있습니다.

<p>$$
U_{\theta}
= \begin{pmatrix}
\cos(\theta) & \sin(\theta)\\[1mm]
\cos(\theta+ \pi/2) & \sin(\theta + \pi/2)
\end{pmatrix}
= \begin{pmatrix}
\cos(\theta) & \sin(\theta)\\[1mm]
-\sin(\theta) & \cos(\theta)
\end{pmatrix}.
$$</p>

이것은 *회전 행렬*의 한 예이며, 특히 원점을 중심으로 $-\theta$ 각도만큼 실수 항목을 가진 2차원 벡터를 회전시킵니다. 우리가 다양한 형태의 회전에 대한 명명 및 매개변수화에 대한 표준 관례를 따른다면, $U_{\theta} = R_y(-2\theta)$를 가지며, 여기서

<p>$$
R_y(\theta) = \begin{pmatrix}
\cos(\theta/2) & -\sin(\theta/2)\\[1mm]
\sin(\theta/2) & \cos(\theta/2)
\end{pmatrix}.
$$</p>

2\. 전략 설명

이제 우리는 양자 전략을 설명할 수 있습니다.

*   **설정:** 앨리스와 밥은 게임을 하나의 얽힘 비트를 공유하며 시작합니다. 앨리스는 큐비트 $\mathsf{A}$를, 밥은 큐비트 $\mathsf{B}$를 보유하며, 두 큐비트 $(\mathsf{X}, \mathsf{Y})$는 $\vert\phi^+\rangle$ 상태에 있습니다.
*   **앨리스의 행동:**
    * 앨리스가 질문 $x = 0$을 받으면, 그녀는 그녀의 큐비트 $\mathsf{A}$에 $U_0$를 적용합니다.
    * 앨리스가 질문 $x = 1$을 받으면, 그녀는 그녀의 큐비트 $\mathsf{A}$에 $U_{\pi/4}$를 적용합니다.

    앨리스가 $\mathsf{A}$에 수행하는 연산은 대안적으로 다음과 같이 설명될 수 있습니다. 앨리스가 이 연산을 적용한 후, 그녀는 **표준 기저 측정으로 $\mathsf{A}$를 측정**하고 그녀의 답변 $a$를 측정 결과로 설정합니다.
*   **밥의 행동:**
    * 밥이 질문 $y = 0$을 받으면, 그는 그의 큐비트 $\mathsf{B}$에 $U_{\pi/8}$를 적용합니다.
    * 밥이 질문 $y = 1$을 받으면, 그는 그의 큐비트 $\mathsf{B}$에 $U_{-\pi/8}$를 적용합니다.

    앨리스에게 했던 것처럼, 우리는 밥의 $\mathsf{B}$에 대한 연산을 다음과 같이 표현할 수 있습니다. 
    
    <p>
    $$
    \begin{cases}
    U_{\pi/8} & \text{if $y = 0$}\\
    U_{-\pi/8} & \text{if $y = 1$}
    \end{cases}
    $$
    </p>

    밥이 이 연산을 적용한 후, 그는 **표준 기저 측정으로 $\mathsf{B}$를 측정**하고 그의 답변 $b$를 측정 결과로 설정합니다.

다음은 이 전략을 설명하는 양자 회로 다이어그램입니다.

<img src="https://quantum.cloud.ibm.com/assets-docs-learning/_next/image?url=%2Flearning%2Fimages%2Fcourses%2Fbasics-of-quantum-information%2Fentanglement-in-action%2Fqiskit-implementation%2FCHSH.avif&w=3840&q=75" style="width:70%">

이 다이어그램에서 우리는 두 개의 일반적인 제어 게이트를 봅니다. 하나는 상단에 있는 $U_{-\pi/8}$에 대한 것이고, 다른 하나는 하단에 있는 $U_{\pi/4}$에 대한 것입니다. 또한 우리는 제어 게이트처럼 보이는 두 개의 게이트를 가지고 있습니다. 하나는 상단에 있는 $U_{\pi/8}$에 대한 것이고, 다른 하나는 하단에 있는 $U_0$에 대한 것인데, 제어를 나타내는 원이 채워져 있지 않습니다. 이것은 **제어가 $0$으로 설정될 때 게이트가 수행되는** (일반적인 제어 게이트처럼 $1$이 아님) 다른 유형의 제어 게이트를 나타냅니다. 따라서 효과적으로 밥은 $y=0$이면 그의 큐비트에 $U_{\pi/8}$를 수행하고 $y=1$이면 $U_{-\pi/8}$를 수행하며, 앨리스는 $x=0$이면 그녀의 큐비트에 $U_0$를 수행하고 $x=1$이면 $U_{\pi/4}$를 수행합니다. 이는 위에서 말로 설명한 프로토콜과 일치합니다.

이제 앨리스와 밥을 위한 이 전략이 얼마나 잘 작동하는지 알아내는 것이 남아 있습니다. 우리는 가능한 네 가지 질문 쌍을 개별적으로 검토하여 이를 수행할 것입니다.

3\. 사례별 분석

*   **사례 1: $(x, y) = (0, 0)$.** \
    이 경우 앨리스는 그녀의 큐비트에 $U_0$를 수행하고 밥은 그의 큐비트에 $U_{\pi/8}$를 수행하므로, 그들이 연산을 수행한 후 두 큐비트 $(\mathsf{A}, \mathsf{B})$의 상태는 다음과 같습니다.

    <p>$$
    \begin{aligned}
    \bigl(U_0 \otimes U_{\pi/8}\bigr) \vert \phi^+\rangle
    & =
    \vert 00 \rangle \langle \psi_0 \otimes \psi_{\pi/8}\vert \phi^+\rangle
    + \vert 01 \rangle \langle \psi_0 \otimes\psi_{5\pi/8}\vert \phi^+\rangle \\
    & \qquad + \vert 10 \rangle \langle \psi_{\pi/2} \otimes \psi_{\pi/8}\vert \phi^+\rangle
    + \vert 11 \rangle \langle \psi_{\pi/2} \otimes \psi_{5\pi/8}\vert \phi^+\rangle\\[2mm]
    & = \frac{
        \cos\bigl(-\frac{\pi}{8}\bigr) \vert 00\rangle
      + \cos\bigl(-\frac{5\pi}{8}\bigr) \vert 01\rangle
      + \cos\bigl(\frac{3\pi}{8}\bigr) \vert 10\rangle
      + \cos\bigl(-\frac{\pi}{8}\bigr) \vert 11\rangle}{\sqrt{2}}.
    \end{aligned}
    $$</p>

    따라서 네 가지 가능한 답변 쌍 $(a, b)$에 대한 확률은 다음과 같습니다.
    <p>
    $$
    \begin{aligned}
    \operatorname{Pr}\bigl((a,b)=(0,0)\bigr) & = \frac{1}{2}\cos^2\Bigl(-\frac{\pi}{8}\Bigr) = \frac{2+\sqrt{2}}{8} \\[2mm]
    \operatorname{Pr}\bigl((a,b)=(0,1)\bigr) & = \frac{1}{2}\cos^2\Bigl(-\frac{5\pi}{8}\Bigr) = \frac{2-\sqrt{2}}{8}\\[2mm]
    \operatorname{Pr}\bigl((a,b)=(1,0)\bigr) & = \frac{1}{2}\cos^2\Bigl(\frac{3\pi}{8}\Bigr) = \frac{2-\sqrt{2}}{8}\\[2mm]
    \operatorname{Pr}\bigl((a,b)=(1,1)\bigr) & = \frac{1}{2}\cos^2\Bigl(-\frac{\pi}{8}\Bigr) = \frac{2+\sqrt{2}}{8}
    \end{aligned}
    $$
    </p>

    그런 다음 합계를 통해 $a = b$일 확률과 $a \neq b$일 확률을 얻을 수 있습니다. 

    <p>$$
    \begin{aligned}
    \operatorname{Pr}(a = b) & = \frac{2 + \sqrt{2}}{4}\\[2mm]
    \operatorname{Pr}(a \neq b) & = \frac{2 - \sqrt{2}}{4}
    \end{aligned}
    $$</p>

    질문 쌍 $(0, 0)$에 대해 앨리스와 밥은 $a = b$이면 승리하며, 따라서 그들은 이 경우 $\frac{2 + \sqrt{2}}{4}$의 확률로 승리합니다.

    <p>$$
    \frac{2 + \sqrt{2}}{4}.
    $$</p>

*   **사례 2: $(x, y) = (0, 1)$.** \
    이 경우 앨리스는 그녀의 큐비트에 $U_0$를 수행하고 밥은 그의 큐비트에 $U_{-\pi/8}$를 수행하므로, 그들이 연산을 수행한 후 두 큐비트 $(\mathsf{A}, \mathsf{B})$의 상태는 다음과 같습니다.

    <p>$$
    \begin{aligned}
    \bigl(U_0 \otimes U_{-\pi/8}\bigr) \vert \phi^+\rangle
    & =
    \vert 00 \rangle \langle \psi_0 \otimes \psi_{-\pi/8}\vert \phi^+\rangle
    + \vert 01 \rangle \langle \psi_0 \otimes\psi_{3\pi/8}\vert \phi^+\rangle \\
    & \qquad + \vert 10 \rangle \langle \psi_{\pi/2} \otimes \psi_{-\pi/8}\vert \phi^+\rangle
    + \vert 11 \rangle \langle \psi_{\pi/2} \otimes \psi_{3\pi/8}\vert \phi^+\rangle\\[2mm]
    & = \frac{
        \cos\bigl(\frac{\pi}{8}\bigr) \vert 00\rangle
      + \cos\bigl(-\frac{3\pi}{8}\bigr) \vert 01\rangle
      + \cos\bigl(\frac{5\pi}{8}\bigr) \vert 10\rangle
      + \cos\bigl(\frac{\pi}{8}\bigr) \vert 11\rangle}{\sqrt{2}}.
    \end{aligned}
    $$</p>

    따라서 네 가지 가능한 답변 쌍 $(a, b)$에 대한 확률은 다음과 같습니다.
    <p>$$
    \begin{aligned}
    \operatorname{Pr}\bigl((a,b)=(0,0)\bigr) & = \frac{1}{2}\cos^2\Bigl(\frac{\pi}{8}\Bigr) = \frac{2+\sqrt{2}}{8} \\[2mm]
    \operatorname{Pr}\bigl((a,b)=(0,1)\bigr) & = \frac{1}{2}\cos^2\Bigl(-\frac{3\pi}{8}\Bigr) = \frac{2-\sqrt{2}}{8}\\[2mm]
    \operatorname{Pr}\bigl((a,b)=(1,0)\bigr) & = \frac{1}{2}\cos^2\Bigl(\frac{5\pi}{8}\Bigr) = \frac{2-\sqrt{2}}{8}\\[2mm]
    \operatorname{Pr}\bigl((a,b)=(1,1)\bigr) & = \frac{1}{2}\cos^2\Bigl(\frac{\pi}{8}\Bigr) = \frac{2+\sqrt{2}}{8}
    \end{aligned}
    $$</p>

    다시 한 번, 합계를 통해 $a = b$일 확률과 $a \neq b$일 확률을 얻을 수 있습니다. 
    <p>$$
    \begin{aligned}
    \operatorname{Pr}(a = b) & = \frac{2 + \sqrt{2}}{4}\\[2mm]
    \operatorname{Pr}(a \neq b) & = \frac{2 - \sqrt{2}}{4}
    \end{aligned}
    $$</p>

    질문 쌍 $(0, 1)$에 대해 앨리스와 밥은 $a = b$이면 승리하며, 따라서 그들은 이 경우 $\frac{2 + \sqrt{2}}{4}$의 확률로 승리합니다.
    <p>$$
    \frac{2 + \sqrt{2}}{4}.
    $$</p>

*   **사례 3: $(x, y) = (1, 0)$.** \
    이 경우 앨리스는 그녀의 큐비트에 $U_{\pi/4}$를 수행하고 밥은 그의 큐비트에 $U_{\pi/8}$를 수행하므로, 그들이 연산을 수행한 후 두 큐비트 $(\mathsf{A}, \mathsf{B})$의 상태는 다음과 같습니다.

    <p>$$
    \begin{aligned}
    \bigl(U_{\pi/4} \otimes U_{\pi/8}\bigr) \vert \phi^+\rangle
    & =
    \vert 00 \rangle \langle \psi_{\pi/4} \otimes \psi_{\pi/8}\vert \phi^+\rangle
    + \vert 01 \rangle \langle \psi_{\pi/4} \otimes\psi_{5\pi/8}\vert \phi^+\rangle \\
    & \qquad + \vert 10 \rangle \langle \psi_{3\pi/4} \otimes \psi_{\pi/8}\vert \phi^+\rangle
    + \vert 11 \rangle \langle \psi_{3\pi/4} \otimes \psi_{5\pi/8}\vert \phi^+\rangle\\[2mm]
    & = \frac{
        \cos\bigl(\frac{\pi}{8}\bigr) \vert 00\rangle
      + \cos\bigl(-\frac{3\pi}{8}\bigr) \vert 01\rangle
      + \cos\bigl(\frac{5\pi}{8}\bigr) \vert 10\rangle
      + \cos\bigl(\frac{\pi}{8}\bigr) \vert 11\rangle}{\sqrt{2}}.
    \end{aligned}
    $$</p>

    따라서 네 가지 가능한 답변 쌍 $(a, b)$에 대한 확률은 다음과 같습니다.

    <p>$$
    \begin{aligned}
    \operatorname{Pr}\bigl((a,b)=(0,0)\bigr) & = \frac{1}{2}\cos^2\Bigl(\frac{\pi}{8}\Bigr) = \frac{2+\sqrt{2}}{8} \\[2mm]
    \operatorname{Pr}\bigl((a,b)=(0,1)\bigr) & = \frac{1}{2}\cos^2\Bigl(-\frac{3\pi}{8}\Bigr) = \frac{2-\sqrt{2}}{8}\\[2mm]
    \operatorname{Pr}\bigl((a,b)=(1,0)\bigr) & = \frac{1}{2}\cos^2\Bigl(\frac{5\pi}{8}\Bigr) = \frac{2-\sqrt{2}}{8}\\[2mm]
    \operatorname{Pr}\bigl((a,b)=(1,1)\bigr) & = \frac{1}{2}\cos^2\Bigl(\frac{\pi}{8}\Bigr) = \frac{2+\sqrt{2}}{8}
    \end{aligned}
    $$</p>

    우리는 다시 한번 $a = b$일 확률과 $a \neq b$일 확률이 다음과 같다는 것을 발견합니다. 
    
    <p>$$
    \begin{aligned}
    \operatorname{Pr}(a = b) & = \frac{2 + \sqrt{2}}{4}\\[2mm]
    \operatorname{Pr}(a \neq b) & = \frac{2 - \sqrt{2}}{4}
    \end{aligned}
    $$</p>

    질문 쌍 $(1, 0)$에 대해 앨리스와 밥은 $a = b$이면 승리하므로, 그들은 이 경우 $\frac{2 + \sqrt{2}}{4}$의 확률로 승리합니다.
    <p>$$
    \frac{2 + \sqrt{2}}{4}.
    $$</p>

*   **사례 4: $(x, y) = (1, 1)$.**
    마지막 경우는 이 경우 승리 조건이 다르기 때문에 예상할 수 있듯이 조금 다릅니다. $x$와 $y$가 모두 $1$일 때, 앨리스와 밥은 $a$와 $b$가 *다를 때* 승리합니다. 이 경우 앨리스는 그녀의 큐비트에 $U_{\pi/4}$를 수행하고 밥은 그의 큐비트에 $U_{-\pi/8}$를 수행하므로, 그들이 연산을 수행한 후 두 큐비트 $(\mathsf{A}, \mathsf{B})$의 상태는 다음과 같습니다.

    <p>$$
    \begin{aligned}
    \bigl(U_{\pi/4} \otimes U_{-\pi/8}\bigr) \vert \phi^+\rangle
    & =
    \vert 00 \rangle \langle \psi_{\pi/4} \otimes \psi_{-\pi/8}\vert \phi^+\rangle
    + \vert 01 \rangle \langle \psi_{\pi/4} \otimes\psi_{3\pi/8}\vert \phi^+\rangle \\
    & \qquad + \vert 10 \rangle \langle \psi_{3\pi/4} \otimes \psi_{-\pi/8}\vert \phi^+\rangle
    + \vert 11 \rangle \langle \psi_{3\pi/4} \otimes \psi_{3\pi/8}\vert \phi^+\rangle\\[2mm]
    & = \frac{
        \cos\bigl(\frac{3\pi}{8}\bigr) \vert 00\rangle
      + \cos\bigl(-\frac{\pi}{8}\bigr) \vert 01\rangle
      + \cos\bigl(\frac{7\pi}{8}\bigr) \vert 10\rangle
      + \cos\bigl(\frac{3\pi}{8}\bigr) \vert 11\rangle}{\sqrt{2}}.
    \end{aligned}
    $$</p>

    따라서 네 가지 가능한 답변 쌍 $(a, b)$에 대한 확률은 다음과 같습니다.

    <p>$$
    \begin{aligned}
    \operatorname{Pr}\bigl((a,b)=(0,0)\bigr) & = \frac{1}{2}\cos^2\Bigl(\frac{3\pi}{8}\Bigr) = \frac{2-\sqrt{2}}{8} \\[2mm]
    \operatorname{Pr}\bigl((a,b)=(0,1)\bigr) & = \frac{1}{2}\cos^2\Bigl(-\frac{\pi}{8}\Bigr) = \frac{2+\sqrt{2}}{8}\\[2mm]
    \operatorname{Pr}\bigl((a,b)=(1,0)\bigr) & = \frac{1}{2}\cos^2\Bigl(\frac{7\pi}{8}\Bigr) = \frac{2+\sqrt{2}}{8}\\[2mm]
    \operatorname{Pr}\bigl((a,b)=(1,1)\bigr) & = \frac{1}{2}\cos^2\Bigl(\frac{3\pi}{8}\Bigr) = \frac{2-\sqrt{2}}{8}
    \end{aligned}
    $$</p>

    확률은 다른 세 가지 경우와 효과적으로 자리를 바꿨습니다. 우리는 합계를 통해 $a = b$일 확률과 $a \neq b$일 확률을 얻습니다. 
    
    <p>$$
    \begin{aligned}
    \operatorname{Pr}(a = b) & = \frac{2 - \sqrt{2}}{4}\\[2mm]
    \operatorname{Pr}(a \neq b) & = \frac{2 + \sqrt{2}}{4}
    \end{aligned}
    $$</p>
    
    질문 쌍 $(1, 1)$에 대해 앨리스와 밥은 $a \neq b$이면 승리하며, 따라서 그들은 이 경우 $\frac{2 + \sqrt{2}}{4}$의 확률로 승리합니다.

    <p>$$
    \frac{2 + \sqrt{2}}{4}.
    $$</p>

그들은 모든 경우에서 $\frac{2 + \sqrt{2}}{4} \approx 0.85$의 동일한 확률로 승리합니다. 

<p>$$
  \frac{2 + \sqrt{2}}{4} \approx 0.85.
$$</p>


따라서 이것이 그들이 전체적으로 승리할 확률입니다. 이는 이 게임에서 어떤 고전적 전략이 할 수 있는 것보다 훨씬 뛰어납니다. 고전적 전략의 승리 확률은 $3/4$로 제한됩니다. 그리고 이것이 이 예시를 매우 흥미롭게 만드는 이유입니다. 

이 확률은 양자 전략에 대한 *최적*의 승리 확률입니다. 우리가 어떤 얽힘 상태나 측정을 선택하더라도 이보다 더 잘할 수는 없습니다. 이 사실은 **치렐손 부등식(Tsirelson's inequality)**으로 알려져 있으며, 이를 처음 증명하고 $\text{CHSH}$ 실험을 게임으로 처음 설명한 보리스 치렐손(Boris Tsirelson)의 이름을 따서 명명되었습니다.

4\. 기하학적 그림

위에 설명된 전략을 기하학적으로 생각하는 것이 가능하며, 이는 앨리스와 밥의 연산을 위해 선택된 다양한 각도 사이의 관계를 이해하는 데 도움이 될 수 있습니다. 

앨리스가 효과적으로 하는 일은 그녀의 질문 $x$에 따라 각도 $\alpha$를 선택하고, 그녀의 큐비트에 $U_{\alpha}$를 적용하고 측정하는 것입니다. 마찬가지로 밥은 $y$에 따라 각도 $\beta$를 선택하고, 그의 큐비트에 $U_{\beta}$를 적용하고 측정합니다. 우리는 $\alpha$와 $\beta$를 다음과 같이 선택했습니다.

<p>$$
\begin{aligned}
\alpha & = \begin{cases}
0 & x=0\\
\pi/4 & x=1
\end{cases}\\[4mm]
\beta & = \begin{cases}
\pi/8 & y = 0\\
-\pi/8 & y = 1
\end{cases}
\end{aligned}
$$</p>

하지만 잠시 동안 $\alpha$와 $\beta$를 임의의 값으로 가정해 봅시다. $\alpha$를 선택함으로써, 앨리스는 효과적으로 다음과 같이 보이는 정규 직교 기저 벡터를 정의합니다. 

<img src="https://quantum.cloud.ibm.com/assets-docs-learning/_next/image?url=%2Flearning%2Fimages%2Fcourses%2Fbasics-of-quantum-information%2Fentanglement-in-action%2Fqiskit-implementation%2Falpha-basis.avif&w=3840&q=75" style="width:60%">

밥도 마찬가지로 수행하지만, 그의 각도는 $\beta$입니다.

<img src="https://quantum.cloud.ibm.com/assets-docs-learning/_next/image?url=%2Flearning%2Fimages%2Fcourses%2Fbasics-of-quantum-information%2Fentanglement-in-action%2Fqiskit-implementation%2Fbeta-basis.avif&w=3840&q=75" style="width:60%">

벡터의 색상은 앨리스와 밥의 답변에 해당합니다. 파란색은 $0$을, 빨간색은 $1$을 나타냅니다. 

이제 우리는 (3)과 (4)를 결합하면 \
 $\langle \psi_{\alpha} \otimes\psi_{\beta} \vert \phi^+ \rangle = \frac{1}{\sqrt{2}} \langle \psi_{\alpha} \vert \psi_{\beta} \rangle$라는 공식을 얻습니다. \
이는 모든 실수 $\alpha$와 $\beta$에 대해 작동합니다.

위에서 우리가 거친 것과 같은 종류의 분석을 따르지만 $\alpha$와 $\beta$를 변수로 사용하여, 우리는 다음을 발견합니다.

<p>$$
  \begin{aligned}
  & \bigl(U_{\alpha} \otimes U_{\beta}\bigr) \vert \phi^+\rangle\\[1mm]
  & \qquad =
    \vert 00 \rangle \langle \psi_{\alpha} \otimes \psi_{\beta}\vert \phi^+\rangle
    + \vert 01 \rangle \langle \psi_{\alpha} \otimes\psi_{\beta + \pi/2}\vert \phi^+\rangle \\
  & \qquad \qquad + \vert 10 \rangle \langle \psi_{\alpha+\pi/2} \otimes \psi_{\beta}\vert \phi^+\rangle
    + \vert 11 \rangle \langle \psi_{\alpha+\pi/2} \otimes \psi_{\beta+\pi/2}\vert \phi^+\rangle\\[2mm]
  & \qquad = \frac{
        \langle \psi_\alpha \vert \psi_\beta \rangle \vert 00\rangle
      + \langle \psi_\alpha \vert \psi_{\beta+\pi/2} \rangle \vert 01\rangle
      + \langle \psi_{\alpha+\pi/2} \vert \psi_\beta \rangle \vert 10\rangle
      + \langle \psi_{\alpha+\pi/2} \vert \psi_{\beta+\pi/2} \rangle \vert 11\rangle
      }{\sqrt{2}}.
  \end{aligned}
$$</p>

우리는 이 두 가지 결론을 내립니다.

<p>$$
\begin{aligned}
\operatorname{Pr}(a = b)  & = \frac{1}{2} \vert \langle \psi_\alpha \vert \psi_\beta \rangle \vert^2
+ \frac{1}{2} \vert \langle \psi_{\alpha+\pi/2} \vert \psi_{\beta+\pi/2} \rangle \vert^2
= \cos^2(\alpha - \beta)\\[2mm]
\operatorname{Pr}(a \neq b) & = \frac{1}{2} \vert \langle \psi_\alpha \vert \psi_{\beta+\pi/2} \rangle \vert^2
+ \frac{1}{2} \vert \langle \psi_{\alpha+\pi/2} \vert \psi_\beta \rangle \vert^2
= \sin^2(\alpha - \beta).
\end{aligned}
$$</p>

이러한 방정식은 앨리스와 밥이 선택한 기저를 겹쳐 놓는다고 상상함으로써 위의 그림과 연결될 수 있습니다.

<p>$$
\cos^2\Bigl(\frac{\pi}{8}\Bigr) = \frac{2 + \sqrt{2}}{4},
$$</p>

반면 그들이 일치하지 않을 확률은 이 각도의 사인 제곱입니다.

$$</p>

</p>

그들이 일치하지 않을 확률은 이 각도의 사인 제곱입니다.

<p>$$

$$</p>

$(x, y) = (1, 0)$일 때, 앨리스와 밥은 $\alpha = \pi/4$와 $\beta = \pi/8$을 선택하고, 그들의 기저를 겹쳐 놓으면 이 그림을 얻습니다.

### 2.5 비고
$\text{CHSH}$ 게임과 같이 얽힘이 순전히 고전적 추론과 일치하지 않는 통계적 결과를 낳는 실험의 기본 아이디어는 벨 상태의 이름이기도 한 존 벨(John Bell)에게서 비롯되었습니다. 이러한 이유로 사람들은 종종 이러한 종류의 실험을 *벨 테스트(Bell tests)*라고 부릅니다. 때때로 사람들은 *벨의 정리(Bell's theorem)*를 언급하기도 하는데, 이는 다양한 방식으로 공식화될 수 있지만, 그 핵심은 양자 역학이 소위 *국소 은닉 변수 이론(local hidden variable theories)*과 양립할 수 없다는 것입니다. $\text{CHSH}$ 게임은 벨 테스트의 특히 깔끔하고 간단한 예시이며, 벨 정리의 증명 또는 시연으로 볼 수 있습니다.

$\text{CHSH}$ 게임은 양자 정보 이론을 실험적으로 테스트하는 방법을 제공합니다. 우리는 $\text{CHSH}$ 게임을 구현하는 실험을 수행하고, 위에 설명된 얽힘 기반 전략을 테스트할 수 있습니다. 이는 얽힘이 실재한다는 높은 수준의 확신을 제공하며, 얽힘을 설명하기 위해 우리가 때때로 생각해내는 모호하거나 시적인 방식과 달리, $\text{CHSH}$ 게임은 얽힘을 *관찰*할 수 있는 구체적이고 테스트 가능한 방법을 제공합니다. 2022년 노벨 물리학상은 이러한 연구 분야의 중요성을 인정합니다. 이 상은 얽힌 광자에 대한 벨 테스트를 통해 얽힘을 관찰한 **알랭 아스펙트(Alain Aspect), 존 클라우저(John Clauser, CHSH의 C), 그리고 안톤 차일링거(Anton Zeilinger)**에게 수여되었습니다.

## 3. Qiskit으로 구현

위에 정의된 양자 전략과 함께 $\text{CHSH}$ 게임을 Qiskit에서 다음과 같이 구현할 수 있습니다.

먼저, 임의의 전략을 인수로 연결할 수 있도록 하는 게임 자체의 정의가 있습니다. 
이 함수는 $\text{CHSH}$ 게임을 실행하며, 전략(두 비트에서 두 비트로 가는 함수)을 사용하여 승리 시 $1$, 패배 시 $0$을 반환합니다.


```python
def chsh_game(strategy):
    # This function runs the CHSH game, using the strategy (a function
    # from two bits to two bits), returning 1 for a win and 0 for a loss.
 
    # Choose x and y randomly
    x, y = random.randint(0, 2), random.randint(0, 2)
 
    # Use the strategy to determine a and b
    a, b = strategy(x, y)
 
    # Decide if the strategy wins or loses
    if (a != b) == (x & y):
        return 1  # Win
    return 0  # Lose
```

이제 앨리스와 밥의 질문에 따라 회로를 출력하는 함수를 만들 것입니다. 단순화를 위해 큐비트에 기본 이름을 사용하도록 할 것이며, 앨리스와 밥의 행동을 위해 내장 $R_y(\theta)$ 게이트를 사용할 것입니다. 이 함수는 위에 설명된 양자 전략을 구현하는 $\text{QuantumCircuit}$을 생성합니다 (얽힘 비트 준비 포함).

```python
def chsh_circuit(x, y):
    # This function creates a `QuantumCircuit` implementing the quantum
    # strategy described above (including the e-bit preparation).
 
    qc = QuantumCircuit(2, 2)
 
    # Prepare an e-bit
    qc.h(0)
    qc.cx(0, 1)
    qc.barrier()
 
    # Alice's actions
    if x == 0:
        qc.ry(0, 0)
    else:
        qc.ry(-pi / 2, 0)
    qc.measure(0, 0)
 
    # Bob's actions
    if y == 0:
        qc.ry(-pi / 4, 1)
    else:
        qc.ry(pi / 4, 1)
    qc.measure(1, 1)
 
    return qc
```

이제 우리는 주어진 입력 쌍 $(x, y)$에 대해 회로를 단 한 번 실행하는 Aer 시뮬레이터를 사용하는 작업을 만들 것입니다. 이 함수는 위에 정의된 적절한 양자 회로를 한 번 실행하고 측정 결과를 반환합니다. `shots=1`로 설정하여 회로를 한 번 실행합니다.

```python
def quantum_strategy(x, y):
    # This function runs the appropriate quantum circuit defined above
    # one time and returns the measurement results
 
    # Setting `shots=1` to run the circuit once
    result = AerSimulator().run(chsh_circuit(x, y), shots=1).result()
    statistics = result.get_counts()
 
    # Determine the output bits and return them
    bits = list(statistics.keys())[0]
    a, b = bits[0], bits[1]
    return a, b
```

우리는 또한 고전적 전략을 정의하고 그것이 얼마나 잘 작동하는지 볼 수 있습니다. 이것은 단 하나의 전략일 뿐이지만 (다른 전략은 코드를 변경하여 테스트할 수 있습니다), 이는 최적의 고전적 전략 중 하나입니다. 

```python
def classical_strategy(x, y):
    # This function implements just one example of an optimal classical
    # strategy for the CHSH game. Other classical strategies can be
    # implemented by changing the bit values assigned to a and b.
 
    # Alice's answer
    if x == 0:
        a = 0
    elif x == 1:
        a = 1
 
    # Bob's answer
    if y == 0:
        b = 1
    elif y == 1:
        b = 0
 
    return a, b
```

다시 1000번 게임을 플레이하여 얼마나 잘 작동하는지 봅시다.
